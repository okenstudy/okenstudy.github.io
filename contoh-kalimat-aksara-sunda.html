Dalam kehidupan sehari-hari, leluhur (karuhun) / nenek moyang menggunakan aksara ini untuk menuliskan suatu pesan yang akan disampaikan kepada pembaca, peninggalan media beraksara Sunda banyak ditemukan dalam bentuk prasasti, naskah lontar, kakawin, dan lain sebagainya

Kalimah "Abdi atoh diajar Bahasa Sunda" bisa diwangun ku wangunan ieu:

Abdi: Aksara A, aksara ba + pamaéh, aksara da + panghulu.Atoh: Aksara A, aksara ta + panolong + pangwisad.Diajar: Aksara da + panghulu, aksara a, aksara ja + panglayar.Bahasa: Aksara ba, Aksara ha, ahsara sa.Sunda: Aksara sa + panyuku, aksara na + pamaéh, aksara da.

AKSARA SWARA

ᮃ (a) ᮅ (u) ᮆ (é) ᮈ (e)ᮄ ( i ) ᮉ (eu) ᮇ (o)

RARANGKEN

|| Rarangken luhur:

Panghulu (a -> i). Contohna: ᮒᮤ // ti.

Pamepet (a -> e). Contohna: ᮒᮨ // te

Paneuleung (a -> eu). Contohna: ᮒᮩ // teu.

Panglayar (+ r). Contohna: ᮒᮁ // tar

Panyecek (+ ng). Contohna: ᮒᮀ // tang

|| Rarangken sejajar:

Panéléng (a -> é). Contohna: ᮒᮦ // té.

Panolong (a -> o). Contohna: ᮒᮧ // to.

Pamingkal (a -> ya). Contohna: ᮒᮡ // tya.

Pangwisad (+ h). Contohna: ᮒᮂ // tah.

Pamaéh (..a -> ..). Contohna: ᮒ᮪ // t.

|| Rarangken handap:

Panyuku (a -> u). Contohna: ᮒᮥ // tu.

Panyakra (a -> ra). Contohna: ᮒᮢ // tra.

Panyiku (a -> la). Contohna: ᮒᮣ // tla.
