5. kpan kita mngunakan user adaministrator dan kpan kita menggunkan user biasa​


Kita perlu menggunakan sudo (superuser), saat kita perlu mengakses sebuah sistem. Misalnya ketika ingin menginstall repo/package, mengontrol bluetooth/wifi, dsb. Dan juga kita perlu sudo saat mengakses file system, seperti grub, source.list, dll.

Kita hanya perlu menggunakan user biasa saat mengakses file end-user (file punya kita), seperti video, foto, musik, dan dokumen. Itu tidak perlu superuser, karena memang yang membuat file itu adalah kita sebagai user biasa, bukan superuser.

Secara umum, perputaran data yaitu : input - proses - output.

Namun pada soal di atas, keterlibatan user menyebabkan data diawali dari user & diakhiri oleh user. Sehingga prosesnya:

- User : memiliki data (yg masih mentah)

- input: data diinput dg alat masukan

- CPU: data diolah dg alat proses

- output: data dioutput dg alat keluaran

- user: mendapatkan data berupa informasi (data yg sudah diolah)

Anonymous, Mempunyai hal keterbatasa hak akses. Keterbatasan yang dimiliki ketika pengguna menggunakan jenis FTP anonymous biasanya meliputi keterbatasan dalam proses akses directrory dan file yang tersedia dalam server yang dituju. Namun sebaliknya ia hanya memiliki kemampuan dalam downloading, baca file tertentu dan pindah directory yang diizinkan oleh pemilik server.


• User legal (authenticated user), adalah sebuah cara lain yang digunakan oleh
pengguna internet dalam mengakses sebuah server dengan menggunakan FTP. Untuk dapat mengkases remote host, cara user legal (authenticated user) menuntut kita untuk memiliki sebuah account khusus yang dimiliki secara pribadi. Cara Kerja Website yaitu … a) URL -> user -> browser -> server -> browser ->user b) browser -> user -> URL -> server -> browser ->user c) user -> browser -> URL -> server -> browser ->user d) user -> browser -> URL -> user -> browser ->user e) user -> browser -> user -> server -> browser ->user


Kelas: IX
Mata pelajaran: Teknologi Informasi
Materi:  Internet

Saya akan mencoba menjawab dengan dua jawaban:

Jawaban pendek:

Cara Kerja Website yaitu …

c) user -> browser -> URL -> server -> browser ->user

Jawaban panjang:

User adalah pengguna komputer. Disini user akan mengakses suatu halaman web di Internet. Karena itu yang user lakukan pertama adalah membuka browser di komputernya.

Browser adalah perangkat lunak yang digunakan untuk mengakses halaman internet melalui jaringan internet. Contoh dari browser adalah Chrome, Firefox, Opera dan Internet Explorer. Untuk menggunakan browser, seorang user akan mengetik alamat URL ke browser.

URL adalah alamat yang digunakan untuk mengakses suatu berkas atau data di jaringan komputer. Alamat internet adalah salah satu bentuk URL. Oleh browser, URL ini akan dikirimkan ke server untuk diproses. Contoh URL misalnya adalah http://www.goole.com. Bila user mengetik alamat ini maka dia akan meminta melalui browser di komputernya untuk menghubungkan diri dengan server Google dan menampilkan halaman utama dari situs Google.

Server adalah sistem komputer yang mengolah alamat internet dan menghantarkan halaman dari data yang disimpan di server kembali untuk ditampilkam di browser pada komputer user yang meminta halaman tersebut.

Halaman web yang dikirmkan bisa berupa halaman statis yang tetap. Misalnya halaman berupa file html atau gambar. Namun halaman web yang dikirimkan bisa juga berupa halaman dinamis yang dihasilkan dari pengolahan database dan dengan menggunakan querry atau pencarian.
