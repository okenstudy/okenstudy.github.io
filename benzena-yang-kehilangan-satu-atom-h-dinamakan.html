8. apabila berat atom C=12 dan H=1 maka bobot karbon dalam 156 gram benzena (C6H6) sebnayak


bab : stoikiometri
kelas : X semester 2
kode : 10. 7. 9
kata kunci : stoikiometri, massa unsur


massa unsur dalam senyawa
misal senyawa AxBy
massa A dalam AxBy = (x. Ar A) / Mr AxBy x massa AxBy

jawab

Mr C6H6 = 6. Ar C + 6. Ar H
= 6. 12 + 6.1
= 72 + 6
= 78


massa C dalam C6H6 = 6. Ar C / Mr x massa C6H6
= 6.12 / 78 x 156 gram
= 72/78 x 156
= 144 gram

24. Benzena merupakan senyawa hidrokarbon yang mengandung 92.3% atom karbon. Jika massa molekul relatif (Mr) benzena adalah 78 dan Ar C = 12, H = 1; tentukan rumus empiris dan rumus molekul benzena

13n = 78
  n = 6

25. Benzena merupakan senyawa hidro karbon yang mengandung 92,3% atom karbon. Jika masa molekul relatif benzena adalah 78 n Ar C=12, H=1; tentukan rumus molekul empiris dan rumus molekul benzena


Atom C = 92.3%
Atom H = 7.7%
Massa Atom C = 92.3 / 12 = 7.7
Massa Atom H = 7.7 / 1 = 7.7

Perbandingan massa C : H = 1 : 1
Maka rumus molekul empiris CH

Mr CH = 13
Mr (CH)x = 78
x = 78 / 13 = 6
Maka rumus molekul = C6H6

27. benzena merupakan senyawa hidrokarbon yang mengandung 92,3 % atombenzena merupakan senyawa hidrokarbon yang mengandung 92,3 % atom karbon. jika massa molekul relatif (Mr) benzena adalah 78 dan Ar C = 12, H = 1 ; tentukan rumus empiris dan rumus molekul benzena


92,3% = [(jumlah atom C x Ar C)/ Mr benzena] x 100%
           = [(jumlah atom C x 12)/ 78] x 100%
jumlah atom C = 6 (pembulatan dari 5,999)

persentase atom H = 100% - 92,3% = 7,7%

untuk mencari jumlah atom H, pake cara yang sama (tapi Ar C diganti dg Ar H & jumlah atom C diganti jumlah atom H) nanti dapet = 6

jadi rumus molekulnya = C6H6
rumus empirisnya = CH
