Jawaban:

Jika PKP dan memiliki API

PPN Masukan = 10% x 250.000.000 = 25.000.000

Pph 22 = 2,5% x 250.000.000 = 6.250.000

Maka jurnalnya

D peralatan kantor Rp250.000.000

D pajak masukan Rp25.000.000

D Pph 22 dibayar dimuka Rp6.250.000

4. jlaskan cara mlakukan lari jrk pendk yg mliputi grkan kaki grakan ayunan ,lgn dan sikap badan​


Gerakan lengan

- lengan membentuk sudut 90°

- telapak tangan dibuka dan di rapat kan

- lengan berayun ke atas dan ke bawah secara bergantian

- lengan berada di samping badan

Gerakan kaki

- Melakukan awalan dengan sikap start jongkok.

- Kaki menolak sekuatnya sampai mengejang lurus menggunakan ujung kaki.

- Lutut diangkat tinggi setinggi panggul.

- Tungkai bawah mengayun ke depan untuk mencapai langkah lebar.

Sikap badan

- badan condong ke depan

- pandangan lurus ke depan,
