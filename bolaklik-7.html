10. Sebuah beban yang digantungkan pada sebuah ayunan sederhana yang panjangnya L cm,kemudian disimpangkan sehingga bergerak bolakbalik dengan periode 0.16 S apabila tali ayunan tersebut dikurangi sebesar 36% dari panjang semula,frekuensi ayunan menjadi ?


GETARAN

• ayunan bandul

L1 = L, T1 = 0,16 s

L2 = 0,64 L, f2 = __?

frekuensi ayunan pada keadaan pertama

f1 = 1 / T1

f1 = 1 / 0,16

f1 = 6,25 Hz

Hubungan frekuensi ayunan bandul dengan panjang tali

f = (1/2π) √(g / L)

frekuensi berbanding terbalik dengan akar panjangnya.

Jika panjang dijadikan 0,64 kali, frekuensi akan menjadi 1/√0,64 = 1/0,8 = 5/4 kali semula atau 5/4 × 6,25 Hz = 7,8125 Hz ← jwb

atau gunakan perbandingan

f2 / f1 = [(1/2π) √(g / L2)] / [(1/2π) √(g / L1)]

f2 / f1 = √L1 / √L2

f2 = √(L1 / L2) • f1

f2 = √(L1 / (0,64 L1) • (6,25 Hz)

f2 = √(1 / 0,64) • (6,25 Hz)

f2 = (1 / 0,8) • (6,25 Hz)

f2 = (10 / 8) • (6,25 Hz)

f2 = 7,8125 Hz ← jwb
