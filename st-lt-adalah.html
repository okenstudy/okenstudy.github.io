15. Diketahui segitiga RST, dengan <S=90°, <T=60°,dan ST =6 cm.Hitunglah a. Keliling segitiga RSTb. ( sin <T )pangkat 2 + ( sin <R) pangkat 2​


SEGITIGA RST

< S = 90°

< T = 60°

< R = 30° (180° - 90° - 60°)

ST = 6 cm

a]

ST = 6 cm

RS/sin < T = ST/sin < R

RS/sin 60° = 6/sin 30°

RS/(1/2 V3) = 6/(1/2)

RS × 1/2 = 6 × 1/2 V3

RS/2 = 3V3

RS = 3V3 × 2

RS = 6V3 cm

RT/sin < S = ST/sin < R

RT/sin 90° = 6/sin 30°

RT/1 = 6/(1/2)

RT = 6 × 2/1

RT = 12 cm

atau

RT^2 = RS^2 + ST^2 (karena siku-siku)

RT^2 = (6V3)^2 + 6^2

RT^2 = 108 + 36

RT^2 = 144

RT = V144

RT = 12 cm

K segitiga RST

= ST + RS + RT

= 6 + 6V3 + 12

= (18 + 6V3) cm

b]

(sin < T)^2 + (sin < R)^2

= (sin 60°)^2 + (sin 30°)^2

= (1/2 V3)^2 + (1/2)^2

= 3/4 + 1/4

= 4/4

= 1
