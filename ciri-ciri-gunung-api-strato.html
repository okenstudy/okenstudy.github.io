2. Sebutkan ciri ciri gunung api strato


Ciri-ciri gunung api strato:

Berbentuk kerucut tinggiTerusun dari banyak lapisan-lapisan lava dan bahan lainDapat menyebabkan letusan yang besarLebih umum dari gunung api perisaiTerbentuk di zona subduksi

Pembahasan:

Gunung api strato atau stratovolcano, juga dikenal sebagai gunung berapi komposit, adalah gunung api berbentuk kerucut yang dibangun oleh banyak lapisan (strata) lava mengeras, tephra, batu apung dan abu.

Tidak seperti gunung api perisai yang landai, stratovolcano ditandai dengan bentuk kerucut tinggi dan curam, seringkali dengan kawah di puncak

Stratovolcano kadang-kadang disebut "gunung berapi komposit" karena struktur pembentuknya berupa komposit atau lapisan-lapisan dari sisa material letusan yang menumpuk.

Gunung api strato adalah salah satu jenis gunung berapi yang paling umum, berbeda dengan gunung berapi perisai yang lebih jarang.

Contoh stratovolcano yang terkenal adalah Krakatau di Indonesia, yang dikenal karena letusan dahsyatnya pada tahun 1883. Letusan ini menyebabkan tsunami di Selat Sunda.

Stratovolcano umumnya terbentuk di zona subduksi, yaitu pertemuan antara dua lempeng tektonik yang bergerak saling mendekat, gerak konvergen. Gunung-gunung berapi ini membentuk rantai di sepanjang batas lempeng tektonik. Misalnya adalah “Pacific Ring of Fire” yang terbentuk di zona subduksi sekitar Lempeng Pasifik.

3. sebutkan ciri-ciri gunung api strato .


Ciri-ciri gunung api strato antara lain berbentuk seperti kerucut, letusan terjadi berkali-kali, memiliki tipe erupsi campuran (eksplosif dan efusif), serta memiliki badan gunung yang berlapis-lapis. Sebagian besar gunung api di Indonesia tergolong gunung api strato, seperti gunung Merapi dan gunung Tangkuban Perahu.

Pembahasan

Bumi merupakan salah satu planet dalam sistem tata surya yang berada di galaksi Bima Sakti. Bumi adalah planet yang memiliki karakteristik yang mendukung adanya kehidupan. Karakteristik tersebut antara lain adalah adanya ketersediaan air yang cukup, adanya tanah, serta lapisan udara yang seimbang. Bumi tersusun atas tiga lapisan utama, yaitu sebagai berikut:

Litosfer, merupakan lapisan terluar bumi yang tersusun atas batuan-batuan. Astenosfer, merupakan lapisan selubung atau mantel bumi yang terletak di bawah lapisan litosfer berupa material cair kental dan berpijar. Barisfer, merupakan lapisan inti bumi yang terdiri atas nikel dan ferum.

Pelajari lebih lanjut tentang lapisan penyusun bumi di:  https://brainly.co.id/tugas/10400061.

Perlu kamu ketahui bahwa lapisan litosfer bumi memiliki kenampakan topografi yang beragam. Keberagaman topografi tersebut dipengaruhi oleh adanya tenaga pembentuk bumi yang terdiri dari:

Tenaga endogen, merupakan tenaga pembentuk bumi yang berasal dari dalam bumi yang bersifat membangun. Tenaga endogen terdiri dari tektonisme, vulkanisme, dan seisme. Tenaga eksogen, merupakan tenaga pembentuk bumi yang berasal dari luar bumi yang cenderung bersifat merusak. Tenaga eksogen terdiri dari pelapukan, pengikisan, dan pengendapan.

Pelajari lebih lanjut tentang tenaga pembentuk bumi di:  https://brainly.co.id/tugas/2897322.

Salah satu gejala yang terjadi akibat adanya tenaga endogen adalah vulkanisme, yaitu aktivitas gunung api yang berupa keluarnya magma ke permukaan bumi (erupsi). Terdapat tiga tipe erupsi, yaitu sebagai berikut:

Erupsi efusif, merupakan erupsi yang berupa lelehan lava sehingga membentuk gunung api tipe perisai. Erupsi eksplosif, merupakan erupsi yang berupa ledakan sehingga membentuk gunung api tipe maar. Erupsi campuran, merupakan erupsi yang berupa lelehan (efusif) dan ledakan (eksplosif) sehingga membentuk gunung api tipe strato.

Berdasarkan kekuatan erupsinya, gunung api diklasifikasikan menjadi tiga bentuk berikut ini:

Gunung api perisai, merupakan gunung api yang memiliki tipe erupsi efusif atau lelehan. Gunung api maar, merupakan gunung api yang memiliki tipe erupsi eksplosif atau ledakan. Gunung api strato, merupakan gunung api yang memiliki tipe erupsi campuran (eksplosif dan efusif).

1. Gunung api strato Gunung api strato berciri bentuknya seperti kerucut. Strato artinya lapisan, oleh karena badan gunung api ini terdiri dari lapisan lapisan lava yang bercampur dengan hasil-hasil vulkanis lainnya seperti debu, pasir, kerikil, dan bom. Campuran yang dikandungnya memungkinkan endapan pada lereng gunung berlapis-lapis sehingga gunung api semakin tinggi menjulang keatas. Sebagian besar gunung api di Indonesia tergolong bentuk gunung api strato.

2. Gunung api maar Kata maar berasal dan bahasa Jerman yang artinya “kawah”. Maar terjadi karena peletusan gunung api itu hanya terjadi satu kali saja. Setelah itu kegiatan vulkanis berhenti sama sekali. Akibat peletusan yang terjadi, terbentuklah lubang besar berbentuk corong, yang dikelilingi oleh tebing yang terombak ketiká terjadi letusan. Apabila dasar dan dinding maar tidak dapat ditembus air, maká terbentuklah danau yang disebut danau maar. Namun, ada juga maar yang kering karena jenis tanah di dasarnya tidak dapat menahan air. Contoh gunung api maar antara lain maar di Gunung Lamongan (Jawa Timur), maar di daerah Pegunungan Eifel (Jerman), maar di Dataran Tinggi Auvergne (Perancis).

3. Gunung api Perisai/Prisma (Tameng)
Gunung api perisai berciri lerengnya agak landai berbentuk perisai. Gunung api ini hanya terdiri dari lapisan-lapisan lava saja, karena lava yang keluar dari gunung api hanya berupa lava yang cair sekali, sehingga dapat mengalir jauh menuruni lereng, kemudian mengalami pembekuan. Gunung api perisai terdapat di Kepulauan Hawai yaitu Gunung Mauna Loa dan Gunung Kilauea. Di Indonesia tidak ditemukan jenis gunung api perisai.

Gunung api strato sering disebut juga dengan gunung api kerucut. Di Indonesia kebanyakan gunung api kerucut. Letusan gunung api kerucut termasuk letusan kecil. 22. Ciri ciri gunung berapi tipe perisai, maar, strato, dan stromboli

Lava/magma sangat encer,Kedalaman dapur magma dangkal.Tekanan gas (viskositas) sedang.

1. Gunung Api Strato atau Kerucut. Kebanyakan gunung berapi didunia merupakan gunung api kerucut. Kerucut ini terbentuk karena materi letusan gunung berapi merupakan campuran antara hasil erupsi efusif dan erupsi eksplosif. Sebagian gunung berapi di Sumatra, Jawa, Bali, Nusa Tenggara dan Maluku termasuk gunung berapi kerucut.

2. Gunung Api Maar. Gunung api maar terbentuk karena adanya letusan eksplosif dari dapur magma yang relative kecil atau dangkal. Contoh gunung api ini antara lain Gunung Bromo dan Gunung Tangkuban Perahu di Indonesia.

3. Gunung Api Perisai. Gunung ini terbentuk karena magma yang keluar dari dapur magma bersifat cair. Di Indonesia tidak ada gunung yang berbentu perisai. Gunung api perisai contohnya Maona Loa Hawaii, Amerika Serikat.

Dan berikut macam-macam tipe gunung api 

1. Stromboli – Tipe ini diakibatkan adanya tekanan erupsi yang tidak terlalu kuat akan tetapi berlangsung lama, hal ini disebabkan oleh magma yang cair, tekanan gasnya sedang dan letak dapur magmanya dangkal. Contoh letusan gunung stromboli adalah Gunung Raung di jawa Timur.

2. Hawaii – Tipe ini dicirikan oleh daya erupsinya yang lemah, antara lain karena lavanya cair, tekanan gasnya rendah serta dapur magmanya dangkal. Bentuk gunung apinya perisai. Contohnya letusan gunung Kilauea, Mauna Kea dan Mauna Loa.

3. Vulkano Lemah – Tipe ini mempunyai ciri tekanan gasnya sedang, dapur magma tidak terlalu dalam. Contohnya tipe ini adalah letusan Gunung Bromo dan Semerudi jawa Timur.

4. Vulkano Kuat – Tipe ini mempunyai ciri tekanan gasnya tinggi, karena dapur magmanya lebih dalam. Contoh tipe ini adalah letusan Gunung Etna di Italia.

5. Tipe Merapi – Tipe ini dicirikan sifat lavanya yang cair kental dan tekanan gas agak rendah. Lava tersebut dikeluarkan dari pipa kepundansangat lambat sehingga membeku dan menjadi sumbat lava. Contohnya Gunung Merapi di Jawa Tengah.

6. Tipe Pelee – Tipe ini mempunyai ciri erupsinya sangat eksplosif karena magmanya sangat kental, tekanan gas tinggi, dan dapur magma yang dalam. Contoh dari tipe ini adalah Gunung Pelee di Amerika Tengah.

7. St Vincent – Tipe ini mempunyai ciri letusannya tidak terlalu kuat. Magmanya sangat kental, dapur magmanya dangkal sehinggatekanan gasnya sedang. Contoh gunung tipe ini adalah St Vincent dikepulauan Antiles dan Gunung Kelud di Indonesia.
