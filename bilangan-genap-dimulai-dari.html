jawab :
untuk menjawabnya cari dahulu banyaknya bilangan genap tersebut , rumusnya :
Un = a+(n-1)b
82 = 2 + (n-1) 2
82 = 2 + 2n -2
82-2+2 = 2n
82 = 2n
n = 82/2 = 41

nah sekarang tinggal cari jumlah bilangan genap mulai dari 2 sampai 82 , rumusnya :
Sn = n/2 (a+Un)
S(41) = 41/2 (2+82)
= 41/2 (84)
= 41 × 42 = 1722

Barisan dan Deret

◾ Point A ◾
Bilangan genap dari 2 sampai 82
2, 4, 6, ..., 82

beda (b) = U2 - U1
b = 4 - 2
b = 2

Cari n-nya dulu
Un = U1 + (n - 1) b
82 = 2 + (n - 1) 2
82 = 2 + 2n - 2
82 = 2n
82/2 = n
n = 41

Jumlah bilangan genap dari 2 sampai 82
Sn = n/2 (U1 + Un)
S41 = 41/2 (2 + 82)
S41 = 41/2 x 84
S41 = 41 x 42
S41 = 1722

Jadi jumlah bilangan genap mulai dari 2 sampai 82 adalah 1722.

◾ Point B ◾
Misalkan,
bilangan pertama = a
bilangan kedua = a + 2 (ditambah dua kerena selisih bil genap yg berurutan adalah dua)

Bil pertama + bil kedua = 650
a + a + 2 = 650
2a = 650 - 2
2a = 648
a = 648/2
a = 324

Jadi bilangan pertama = 324
bilangan kedua = 324 + 2 = 326

25. A. Tentukan jumlah bilangan genap mulai dari 2 sampai dengan 82! B. Jumlah bilangan genap berurutan adalah 650. Tentukan banyak bilangan genap itu! #Cara&RumusTQ


A. Tentukan jumlah bilangan genap mulai dari 2 sampai dengan 82!
B. Jumlah bilangan genap berurutan adalah 650. Tentukan banyak bilangan genap itu!

Pembahasan :

A. 2 + 4 + 6 + 8 + .... + 82
a = 2
b = 2
Un = 2n

Un = 82
2n = 82
n = 41

Jumlah semua bilangan genap dari 2 sampai 82 adalah
Sn = n/2 (a + Un)
S₄₁ = 41/2 (2 + 82)
S₄₁ = 41/2 (84)
S₄₁ = 41 (42)
S₄₁ = 1.722


B. 2 + 4 + 6 + .... + 2n = 650
n/2 (a + Un) = 650
n/2 (2 + 2n) = 650
n + n² = 650
n² + n - 650 = 0
(n + 26)(n - 25) = 0
n = -26 atau n = 25

Jadi banyaknya bilangan genap itu ada 25 bilangan
