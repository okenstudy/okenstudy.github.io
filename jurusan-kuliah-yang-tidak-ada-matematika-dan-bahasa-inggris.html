20. Dari 100 orang mahasiswa,40 mahasiswa mengikuti kuliah bahasa Inggris,50 mahasiswa mengikuti kuliah statistika,35 mahasiswa mengikuti kuliah matematika dasar,30 mahasiswa mengikuti kuliah bahasa Inggris dan statistika,15 mahasiswa mengikuti kuliah bahasa Inggris dan matematika dasar,10 mahasiswa mengikuti kuliah statistika dan matematika dasar dan 5 mahasiswa mengikuti kuliah ketiganya. Berapa banyak mahasiswa yang tidak mengikuti kuliah ketiga-tiganya?


Diketahui :

Jumlah = 100

Bahasa inggris = 40

Statistika = 50

Matematika = 35

Bahasa inggris dan statistika = 30

Bahasa Inggris dan matematika = 15

Statistika dan matematika = 10

ketiganya = 5

kita umpamakan :

Jumlah = S = 100

ketiganya = K = 5

Bahasa Inggris dan statistika = BS + K = 30

Bahasa Inggris dan matematika = BM + K = 15

Statistika dan matematika = SM + K = 10

Bahasa Inggris = B + BS + BM + K = 40

Statistika = S + BS + SM + K = 50

Matematika = M + BM + SM + K = 35

Tidak ketiganya = T

Langkah penyelesaian :

S = B + S + M + BS + BM + SM + K + T

100 = B + S + M + BS + BM + SM + K + T

kedua ruas ditambah 2k

100 + 2K = B + S + M + BS + BM + SM + K + T + 2K

100 + 2(5) = B + S + M + BS + BM + SM + 3K + T

110 = B + S + M + (BS + K) + (BM + K) + (SM + K) + T

110 = B + S + M + 30 + 15 + 10 + T

110 = B + S + M + T + 55

55 = B + S + M + T

kedua ruas ditambah 2(BS+BM+SM)+3K

55 + 2(BS + BM + SM) + 3K

= B + S + M + T + 2(BS + BM + SM) + 3K

55 + 2BS + 2BM + 2SM + 3K

= B + S + M + T + 2BS + 2BM + 2SM + 3K

55+(BS + K)+(BM + K)+(SM + K)+(BS + BM + SM)

= (B+BS+BM+K)+(S+BS+SM+K)+(M+BM+SM+K)+T

55+30+15+10+(BS + BM + SM) = 40+50+35+T

110+(BS + BM + SM) = 125 + T

BS + BM + SM = 15 + T

kedua ruas ditambah 3K

BS + BM + SM + 3K = 15 + T + 3K

(BS + K) + (BM + K) + (SM + K) = 15 + T + 3(5)

30 + 15 + 10 = 15 + T + 15

T = 15 + 25

T = 30

Jadi siswa yg tidak mengikuti ketiganya adalah 30 orang
