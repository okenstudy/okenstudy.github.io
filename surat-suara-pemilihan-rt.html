28. Desa suka maju sedang mengadakan pemilihan calon rt, calon rt a mendapat suara ½ lebih banyak dari calon rt b, dan calon rt b mendapat 2/4 lebih sedikit dari calon rt c. jika calon rt c mendapat 652 suara, berapakah jumlah suara yang didapat calon rt a?


Jawaban:

Ada di penjelasan

Penjelasan dengan langkah-langkah:

jumlah suara calon rt a = (1/2 x jumlah suara calon rt b) + jumlah suara calon rt b

jumlah suara calon rt b = jumlah suara calon rt c - (2/4 x jumlah suara calon rt c)

jumlah suara calon rt c = 652 suara

jumlah suara calon rt b = jumlah suara calon rt c - (2/4 x jumlah suara calon rt c) = 652 - (2/4 x 652) = 652 - 326 = 326 suara
