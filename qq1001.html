2. jadikan bilangan biner berikut menjadi bilangan desimal : 1. 110011= .... 2. 011001= .... 3. 111001= .... 4. 101010= .... 5. 111100= ....​


Jawaban singkat:

1. 110011 (biner) = 51 (desimal)2. 011001 (biner) = 25 (desimal)3. 111001 (biner) = 57 (desimal)4. 101010 (biner) = 42 (desimal)5. 111100 (biner) = 60 (desimal)

Penjelasan panjang:

Perhitungan konversi biner ke desimal:

1. 110011 = 1x2⁵+1x2⁴+0x2³+0x2²+1x2¹+1x2⁰ = 32+16+0+0+2+1 = 51

Jadi 110011 (biner) = 51 (desimal)

2. 011001 = 0x2⁵+1x2⁴+1x2³+0x2²+0x2¹+1x2⁰ = 0+16+8+0+0+1 = 25

Jadi 011001 (biner) = 25 (desimal)

3. 111001 = 1x2⁵+1x2⁴+1x2³+0x2²+0x2¹+1x2⁰ = 32+16+8+0+0+1 = 57

Jadi 111001 (biner) = 57 (desimal)

4. 101010 = 1x2⁵+0x2⁴+1x2³+0x2²+1x2¹+0x2⁰ = 32+0+8+0+2+0 = 42

Jadi 101010 (biner) = 42 (desimal)

5. 111100 = 1x2⁵+1x2⁴+1x2³+1x2²+0x2¹+0x2⁰ = 32+16+8+4+0+0 = 60

Jadi 111100 (biner) = 60 (desimal)

29. 11001 biner ke oktal​


Jawaban singkat:

11001 (biner) = 31 (oktal)

Penjelasan tambahan:

Aturan konversi biner (basis 2) ke oktal (basis 8):

- kelompokkan biner jadi 3 digit (dari kanan)

- ubah bilangan biner jadi oktal dg referensi tabel sistem bilangan

Berdasarkan soal, maka pengelompokan binernya menjadi

11 (biner) = 3 (oktal)

001 (biner) = 1 (oktal)

Jadi kesimpulannya 11001 (biner) = 31 (oktal)

# tabel sistem bilangan dapat dilihat pada lampiran
