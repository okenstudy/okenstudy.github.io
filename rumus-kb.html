13. jika larutan mendidih pada suhu 100,104 celcius Kb= 0,52 Kf= 1,86. tentukan titik beku larutanya pake rumus yang mana? help


Diket:
Tb larutan = 100,104 'C
Kb = 0,52 C/molal
Kf = 1,86 C/molal
Dit: Tf larutan...?
Cari ΔTb
ΔTb = Tb larutan - Tb pelarut
ΔTb = 100,104 - 100
ΔTb = 0,104 'C
cari m dr ΔTb
ΔTb = m x Kb
0,104 = m x 0,52
m = 0,104/0,52
m = 0,2 molal
cari ΔTf
ΔTf = m x Kf
ΔTf = 0,2 x 1,86
ΔTf = 0,372
maka:
Tf = ΔTf + Tfo
Tf = 0,372 + 0
Tf = 0,372 'C
titik beku larutan = 0,372'C
∆Tb = Tb larutan - Tb pelarut
= 100,104 - 100
= 0,104

∆Tb = Kb.m
0.104 = 0,52. m
m = 0,2

∆Tf = Kf. m
= 1,86 . 0,2 = 0,372

∆Tf = Tf pelarut - Tf larutan
0,372 = 0 - Tf larutan
Tf larutan = -0,372
