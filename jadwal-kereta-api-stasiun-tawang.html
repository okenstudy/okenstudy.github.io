11. berdasarkan jadwal kedatangan di stasiun A, kereta api seharusnya datang pada pukul 07.00 WIB. pada pukul 06.30 WIB,kerta api masih berada di stasiun B yang berjarak 60 km dari stasiun A. berapakah kelajuan rata-rata agar kereta tepat waktu sampai di stasiun A?


diketahui

jarak antara stasiun A dan B adalah 60 km, dan waktu yang tersisa adalah 30 menit.

ditayakan

kelajuan rata rata ( jarak/waktu)?

jawaban

kelajuan bukan merupakan sebuah vektor, dan kelajuan rata rata merupakan kelajuan tetap. maka yang harus kita lakukan adalah membagi jarak oleh waktu yaitu

30 menit = 1/2 jam

60 : 1/2 jam = 120 km/h

[penggunaan satuan kelajuan/kecepatan umumnya adalah km/hours, atau m/second]

waktu datang 07:00
waktu brngkat 06:30
jarak 60 km

waktu tempuh = waktu datang - waktu berangkat
= 07.00 - 06.30
= 30 menit
= ½ jam

kelajuan = jarak tempuh / waktu
= 60 / ½
= 120 km/ jam.

jadi kelajuan kereta agar datang ke stasiun A tepat waktu adalah 120 km/jam

18. Kereta api joglosemarkerto berangkat dari stasiun balapan solo pada pukul 18.20 menuju stasiun semarang tawang yang berjarak 108 km. akumulasi waktu berhenti kereta api pada beberapa stasiun sebelum stasiun semarang tawang adalah 40 menit. jika kereta api tiba di stasiun semarang tawang pada pukul 21.00, kecepatan rata-rata kereta api terebut adalah…km/jam


Rumus antara jarak, waktu, dan kecepatan yaitu s = v.t. Sehingga kecepatan kereta = 54 km/jam.

Penjelasan dengan langkah-langkah:

Diketahui:

Kereta api joglosemarkerto berangkat dari stasiun balapan solo pada pukul 18.20 menuju stasiun semarang tawang yang berjarak 108 km. akumulasi waktu berhenti kereta api pada beberapa stasiun sebelum stasiun semarang tawang adalah 40 menit.

Ditanya:

jika kereta api tiba di stasiun semarang tawang pada pukul 21.00, kecepatan rata-rata kereta api terebut adalah…km/jam?

Jawab:

Rumus antara jarak, waktu, dan kecepatan yaitu:

s = v.t

Dengan s adalah jarak, v adalah kecepatan, dan t adalah waktu.

Waktu kereta berjalan = 21.00 - 18.20 - 00.40 = 2 jam

Kecepatan kereta = 108 km : 2 jam = 54 km/jam

19. Sebuah kereta api meninggalkan stasiun 1 jam sebelum waktu yang dijadwalkan. Masinisnya menurunkan kecepatannya 20km/h. Di stasiun berikutnya, sejauh 120 km, kereta itu sampai tepat waktu sesuai jadwal. Kecepatan awal kereta api sebelum dilambatkan adalah (km/h)


Kecepatan awal = v1
waktu tempuh dg v1 = t1

kecepatan yg dipake = v2 = 20 km/h
t2 = (t1 + 1) jam

s = 120 km
s1 = v1 . t1 = 120 km

s2 = v2 × t2 = 120
120 = 20 × (t1 + 1)
t1 + 1 = 120/20
t1 = 6 - 1
t1 = 5 jam

S1 = v1 × t1 = 120
v1 = 120/t1
v1 = 120/5
v1 = 24 km/h

Jadi kecepatan awal KA sebelum dilambatkan = v1 = 24 km/h
