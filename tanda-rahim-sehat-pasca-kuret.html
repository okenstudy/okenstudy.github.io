14. di bawah ini bukan tanda gejala pasca vulkanik yaitu...


Di bawah ini yang bukan tanda gejala pasca vulkanik yaitu terjadinya gempa tremor. Hal ini dikarenakan gempa tremor merupakan salah satu tanda bahwa gunung api akan erupsi. Sedangkan gejala pasca vulkanik merupakan gejala-gejala yang timbul setelah adanya erupsi gunung api. Gejala pasca vulkanik diantaranya adalah munculnya geyser (mata air panas), terbentuknya kawah, munculnya sumber gas belerang, dan lain sebagainya.

Pembahasan

Bumi merupakan salah satu planet dalam sistem tata surya yang berada di galaksi Bima Sakti. Bumi adalah planet yang memiliki karakteristik yang mendukung adanya kehidupan. Karakteristik tersebut antara lain adalah adanya ketersediaan air yang cukup, adanya tanah, serta lapisan udara yang seimbang. Bumi tersusun atas tiga lapisan utama, yaitu sebagai berikut:

Litosfer, merupakan lapisan terluar bumi yang tersusun atas batuan-batuan. Astenosfer, merupakan lapisan selubung atau mantel bumi yang terletak di bawah lapisan litosfer berupa material cair kental dan berpijar. Barisfer, merupakan lapisan inti bumi yang terdiri atas nikel dan ferum.

Pelajari lebih lanjut tentang lapisan penyusun bumi di: https://brainly.co.id/tugas/758367.

Perlu kamu ketahui bahwa lapisan litosfer bumi memiliki kenampakan topografi yang beragam. Keberagaman topografi tersebut dipengaruhi oleh adanya tenaga pembentuk bumi yang terdiri dari:

Tenaga endogen, merupakan tenaga pembentuk bumi yang berasal dari dalam bumi yang bersifat membangun. Tenaga endogen terdiri dari tektonisme, vulkanisme, dan seisme. Tenaga eksogen, merupakan tenaga pembentuk bumi yang berasal dari luar bumi yang cenderung bersifat merusak. Tenaga eksogen terdiri dari pelapukan, pengikisan, dan pengendapan.

Pelajari lebih lanjut tentang tenaga pembentuk bumi di: https://brainly.co.id/tugas/1790178.

Salah satu gejala yang terjadi akibat adanya tenaga endogen adalah vulkanisme, yaitu aktivitas gunung api yang berupa keluarnya magma ke permukaan bumi (erupsi). Terdapat tiga tipe erupsi, yaitu sebagai berikut:

Erupsi efusif, merupakan erupsi yang berupa lelehan lava sehingga membentuk gunung api tipe perisai. Erupsi eksplosif, merupakan erupsi yang berupa ledakan sehingga membentuk gunung api tipe maar. Erupsi campuran, merupakan erupsi yang berupa lelehan (efusif) dan ledakan (eksplosif) sehingga membentuk gunung api tipe strato.

Material vulkanik yang dikeluarkan ketika terjadi erupsi gunung api terdiri dari beberapa material berikut ini:

Eflata (material padat), terdiri dari lapili, kerikil, pasir, dan debu. Efusifa (material cair), terdiri dari lava dan lahar. Ekshalasi (materisal gas), terdiri dari mofet, solfatara, dan fumarol.

24. Apa perbedaan tanda-tanda gunung api akan meletus dan pasca gunung api (gunung api akan berhenti)?


Taukah kalian bahwa gunung api akan terus beraktivitas (aktif) selama dapur magma masih terus aktif. Sehingga perbedaan antara gunung api mati dengan gunung api aktif  yakni pada aktivitas dapur magma atau yang dikenal dengan batholit. Yuk simak penjelasannya!

Pembahasan  

Vulkanisme merupakan segala aktivitas kegunung apian seperti  erupsi (keluarnya magma ke permukaan bumi) maupun intrusi (menyusupnya magma ke lapisan bumi). Berdasarkan aktivitas magmanya terdapat 2 jenis gunung api yaitu

1. Gunung Api Aktif

Gunung api aktif merupakan sebutan untuk gunung api yang masih berpotensi meledak atau mengalami erupsi. Contohnya Gunung Sinabung, Gunung Seulawah Agam, dan Gunung  Kerinci. Terdapat beberapa ciri-ciri gunung api akan meletus yaitu

Sering terjadi gempa vulkanik

Temperatur di sekitar gunung meningkat

Flora menjadi layu dan mati

Fauna turun ke dataran rendah atau berpindah tempt (bermigrasi)

Munculnya geyser atau sumber mata air panas.

2. Gunung Api Mati

Gunung api mati merupakan sebutan untuk gunung api yang telah rusak dapur magmanya sehingga tidak mungkin berpotensi lagi adanya aktivitas vulkanik.
