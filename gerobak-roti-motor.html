9. pak Roni seorang pedagang roti. Pak Roni menjual roti manis dan roti tawar menggunakan gerobak yang hanya dapat memuat 120 bungkus roti. harga pembelian roti manis sebesar Rp6. 000,00 per bungkus dan roti tawar sebesar Rp4. 000,00 per bungkus. Dari penjualan roti, pak Roni mendapat keuntungan Rp1.000,00 Dari sebungkus roti tawar. jika modal yang dimiliki pak Roni sebesar Rp600.000,00, keuntungan maksimum yang di peroleh sebesar...


Kategori Soal : Matematika - Program Linear
Kelas : XII (3 SMA)
Pembahasan :
Soal belum lengkap. Mari kita tulis dengan benar soalnya.
Pak Roni seorang pedagang roti yang menjual roti manis dan roti tawar dengan menggunakan gerobak. Gerobak tersebut hanya dapat memuat 120 bungkus roti. Harga pembelian roti manis sebesar Rp6.000,00 per bungkus dan roti tawar sebesar Rp4.000,00 per bungkus dengan modal yang dimiliki pak Roni sebesar Rp600.000,00. Penjualan roti tersebut Pak Roni mendapat keuntungan Rp3.000,00 untuk sebungkus roti manis dan Rp1.000,00 untuk sebungkus roti tawar. Berapa keuntungan maksimum yang diperoleh Pak Roni?

Jawab :
Misalkan banyak roti manis = x, banyak roti tawar = y, dan P = pendapatan. Dari data di atas kita dapat menyusun model matematika sebagai berikut.
Gerobak tersebut hanya dapat memuat 120 bungkus roti.
x + y ≤ 120

Harga pembelian roti manis sebesar Rp6.000,00 per bungkus dan roti tawar sebesar Rp4.000,00 per bungkus dengan modal yang dimiliki pak Roni sebesar Rp600.000,00
6.000x + 4.000y ≤ 600.000
⇔6x + 4y ≤ 600
⇔3x + 2y ≤ 300

Karena x merupakan banyaknya roti manis dan y banyaknya roti tawar, maka x dan y adalah bilangan bulat yang tidak negatif (bilangan cacah).
x ≥ 0, y ≥ 0, dengan x, y ∈ C.
fungsi biaya sebuah roti manis dan sebuah roti tawar dapat ditulis sebagai berikut.
memaksimalkan fungsi tujuan P(x,y) = 3.000x + 1.000y.

Gambar terlampir.
Koordinat titik potong dapat ditentukan sebagai berikut. Pertidaksamaan linear kita ubah menjadi persamaan linear, sehingga diperoleh sistem persamaan linear dua variabel :
x + y = 120
3x + 2y = 300
Kita selesaikan dengan metode eliminasi dan substitusi.
x + y = 120      |.2| 2x + 2y = 240
3x + 2y = 300 |.1|  3x + 2y = 300
                             ____________-
                              - x = -60 ⇔ x = 60
KIta substitusikan x = 60 ke persamaan 
x + y = 120
⇔ 60 + y = 120
⇔ y = 120 - 60
⇔ y = 60

Koordinat titik potongnya adalah (60, 60).

Keuntungan maksimum f(x, y) = 3.000x + 1.000y
f(0, 120) = 0 + 120 x 1.000 = 120.000
f(120, 0) = 3.000 x 120 + 0 = 360.000
f(0, 150) = 0 + 150 x 1.000 = 150.000
f(100, 0) = 3.000 x 100 + 0 = 300.000
f(60, 60) = 60 x 3.000 + 60 x 1.000 = 180.000 + 60.000 = 240.000

Jadi, keuntungan maksimumnya adalah Rp360.000,00.

Semangat!

12. Diketahui : Seorang pedagang roti membeli roti I Rp 4.000,00 per bungkus, roti II Rp 2.000,00 per bungkus, dan ia harus membayar Rp 1.600.000,00. Jika gerobaknya dapat menampung 500 bungkus roti, maka banyaknya roti I dan II yang dapat dibeli pedagang tersebut adalah .

4.000x + 2.000y = 1.600.000

4.000x + 2.000y = 1.600.000
4x + 2y = 1.600
2x + y = 800

y = 800 - 2x

x + y = 500
x + 800 - 2x = 500
-x = 500 - 800
-x = -300
x = 300

x + y = 500
300 + y = 500
y = 500 - 300
y = 200

Jadi banyaknya roti yang bisa dibeli adalah roti A 300 buah dan roti B 200 buah

Nah, langkah pertama untuk menyelesaikan permasalahan di atas adalah dengan melakukan permisalan.

Misalkan:

2. banyak roti B yang dibeli oleh pedagang adalah y

Langkah kedua adalah membuat model matematika berdasarkan permisalan di atas dan informasi dalam soal.

1. harga roti A dan B berturut-turut adalah Rp2.000,00 dan Rp1.000,00 per bungkus, sedangkan jumlah uang yang dikeluarkan oleh pedagang untuk membeli roti A sebanyak x bungkus dan roti B sebanyak y bungkus adalah Rp8.000.000,00 → 2.000x + 1.000y = 8.000.000 atau 2x + y = 8.000

2. semua roti yang dibeli oleh pedagang ditaruh di dalam gerobak, sedangkan daya tampung gerobak untuk menampung roti A sebanyak x bungkus dan roti B sebanyak y bungkus adalah 500 → x + y = 500

Langkah ketiga adalah menentukan nilai x dan y dari sistem persamaan linear dua variabel berikut:    2x + y = 8.000    x + y = 500

Nah, karena yang diminta adalah penyelesaian dengan menggunakan determinan matriks, maka ada beberapa langkah yang perlu ditempuh.

① ubah sistem persamaan linear dua variabel di atas ke dalam persamaan matriks AB = C            

A = 2  1
       1  1

B = x
      y

C = 8.000
       500

② buat matriks D dengan cara mengganti semua elemen pada kolom pertama dari matriks A dengan elemen pada matriks C
③ buat matriks E dengan cara mengganti semua elemen pada kolom kedua dari matriks A dengan elemen pada matriks C
④ tentukan determinan dari matriks A, D, dan E

Berdasarkan metode determinan matriks, nilai x diperoleh dengan menggunakan rumus: x = det (D) / det (A), sedangkan nilai y diperoleh dengan menggunakan rumus: y = det (E) / det (A).

22. Umar Bakri adalah pedagang roti. ia menjual roti menggunakan gerobak yg hanya dapat memuat 600 roti. Roti yg dulu jualnya adalah roti manis dan Roti tawar dengan harga masing-masing rp.5.500,00 dan rp.4.500,00 per bungkusnya. Dari penjualan roti-roti ini, ia memperoleh keuntungan rp.500,00 dari sebungkus roti manis dan rp.600,00 dari sebungkus roti tawar. Jika modal yg di miliki Umar Bakri rp.600,000,00, buatlah model matematika dengan tujuan untuk memperoleh keuntungan sebesar-besarnya?


Program linear adalah metode penentuan nilai optimum dari suatu persoalan linear. Nilai optimum (maksimum atau minimum) diperoleh dari nilai dalam suatu himpunan penyelesaiaan persoalan linear. Di dalam persoalan linear terdapat fungsi linear yang disebut sebagai fungsi objektif. Persyaratan, batasan, dan kendala dalam persoalan linear merupakan sistem pertidaksamaan linear.

Pembahasan

Misal, x menyatakan roti manis, dan y menyatakan roti tawar.

Permasalahan tersebut ditulis ke dalam model matematikanya yaitu

x + y ≤ 600

x ≥ 0

y ≥ 0

F(x,y) = 500x + 600y

Pertidaksamaan tersebut digambar grafiknya dan didapatkan titik-titik pojoknya, yaitu A(0,600), O(0,0), dan B(600,0).

Masing-masing titik pojok disubstitusikan ke dalam F(x,y) = 500x +600y

A(0,600) ⇒ 500(0) +600(600) = 3.600.000

O(0,0) ⇒ 500(0) +600(0) = 0

B(600,0) ⇒ 500(600) +600(0) = 3.000.000

Jadi, keuntungan maksimumnya yaitu Rp 3.600.000 dengan menjual roti tawar sebanyak 600 bungkus.

y = roti tawar

5.000x + 4.500y < 600.000

roti manis : 500x 
roti tawar : 600y

f(x,y) = 500x + 600y
