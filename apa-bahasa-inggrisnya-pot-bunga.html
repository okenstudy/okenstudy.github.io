13. Kuis [+50]: Di lapangan ditaruhlah pot-pot bunga. Di baris pertama ditaruhlah pot bunga sebanyak 91 pot bunga, baris ketiga ada 85 pot bunga. [a] Di baris berapa terdapat 1 pot bunga saja? [b] Berapakah jumlah semua pot bunga yang ditaruh di lapangan?


Ini adalah Barisan Aritmatika.

Setiap baris berkurang 6 Pot Bunga.

Rumus suku ke n nya :

Un = a + (n-1) x b

U1 = a = 91

U3 = 85

U3 = a + 2b

a + 2b = 85

91 + 2b = 85

2b = -6

b = -3

Maka Baris yang terdapat 1 Pot Bunga :

1 = 91 + (n-1) x (-3)

1 = 91 - 3n + 3

1 = -3n + 94

3n = 93

n = 93/3 = 31

Jawaban A = Baris ke 31

B. Jumlah Pot Bunga = Sampai suku ke 31 atau sisa 1 Pot Bunga.

Sn = n/2 (a + a + (n-1) x b)

S31 = 31/2 (91 + 91 + 30 x (-3))

S31 = 15,5 (182 + (-90))

S31 = 15,5 x 92

S31 = 1426 Pot Bunga Jawabannya.

Di  setiap baris  terdapat 1 pot bunga saja terdapat 31

jumlah semua pot bunga yang ditaruh di lapangan sebanyak 1.426

Diketahui :

Di lapangan ditaruhlah pot-pot bunga. Di baris pertama ditaruhlah pot bunga sebanyak 91 pot bunga, baris ketiga ada 85 pot bunga.

Ditanya :

[a] Di baris berapa terdapat 1 pot bunga saja?

[b] Berapakah jumlah semua pot bunga yang ditaruh di lapangan?

Penyelesaian :

Barisan pertama = 91

Barisan ke 3 = 85

Rumus barisan arimatika :

Sn = n : 2 x { a + a + (n - 1 ) x b }

Un = a + ( n - 1 ) x b

U1 / a

b = selisih setiap pola

Un = a + 2b

U3 = a + 2b

U85 = a + 2b

U85 = 91 + 2b

2b = 85 - 91

2b = - 6

b = - 6 : 2

b = - 3

Massukan b = 3 , ke rumus Un arimatika

Un = a + ( n - 1 ) x b

1 = 91 + ( n - 1 ) x ( - 3)

1 = 91 + ( - 3n + 3 )

1 = 91 - 3n +  3

1 = - 3n + 94

- 3n = - 94 + 1

- 3n = - 93

n = - 93 : ( - 3 )

n = 31

Berapakah jumlah semua pot bunga yang ditaruh di lapangan?

Sn = n : 2 x { a + a + ( n - 1 ) x b }

S31 = 31 : 2 x { 91 + 91 + (  31 - 1 ) x ( - 3 )

S31 = 15.5 { 91 + 91 + 30 x ( - 3 ) }

S31 = 15.5 x { 91 + 91 - 90 }

16. Ciri ciri pot bunga dalam bahasa inggris


A flowerpot, flower pot, or plant pot is a container in which flowers and other plants are cultivated and displayed.

Ciri - cirinya(the characteristic of flower pot) :
1. Historically, and still to a significant extent today, they are made from terracotta.
2. Flowerpots are now often also made from plastic, wood, stone, or sometimes biodegradable material.
3. An example of biodegradable pots are ones made of heavy brown paper, cardboard, or peat moss in which young plants for transplanting are grown.
4. For seedling starting in commercial greenhouses or polytunnels, pots usually take the form of trays with cells, each cell acting as one small pot. These trays are often called flats.
5. There are often holes in the bottom of pots, to allow excess water to flow out, sometimes to a saucer that is placed under the flowerpot. The plant can use this water with its roots, as needed.
