12. Kéreta api Fajar Utama berangkat dari Jogja ke Purworejo pukul 07.00 dengan kecepatan ra - rata 64 km/jam. Pada saat yang sama, kereta Prameks berangkat dari Purworejo menu Yogyakarta dengan kecepatan rata-rata 56 km/jam. Jika jarak Yogyakarta - Purworejo 60 k maka kereta Prameks akan berpapasan dengan kereta Fajar Utama setelah berjalan sejauh km dari Purworejo. ​


Untuk menentukan jarak dari Purworejo dimana kereta Prameks berpapasan dengan kereta Fajar Utama, perlu diketahui waktu yang dibutuhkan oleh kedua kereta tersebut untuk mencapai titik tersebut. Dapat digunakan rumus: jarak = kecepatan x waktu. Karena kedua kereta berangkat pada waktu yang sama dan memiliki jarak yang sama antara kedua stasiun, maka waktu yang dibutuhkan oleh kedua kereta untuk mencapai titik berpapasan sama.

Jika kereta Fajar Utama berangkat dari Jogja pukul 07.00 dengan kecepatan rata-rata 64 km/jam, maka waktu yang dibutuhkan untuk mencapai Purworejo adalah 60 km / 64 km/jam = 0,9375 jam atau 56,25 menit.

Sedangkan jika kereta Prameks berangkat dari Purworejo pukul 07.00 dengan kecepatan rata-rata 56 km/jam, maka waktu yang dibutuhkan untuk mencapai Yogyakarta adalah 60 km / 56 km/jam = 1,0714 jam atau 64,28 menit.

Jadi kedua kereta akan berpapasan setelah 56,25 menit dari Purworejo.

Jarak yang ditempuh kereta Prameks dari purworejo saat berpapasan dengan kereta Fajar Utama adalah 56 km/jam x 56,25 menit = 3168,75 km.

Penjelasan dengan langkah-langkah:

Waktu dalam satuan SI (Satuan Internasional) adalah sekon atau biasa disebut detik.

1 jam = 3600 detik

10 menit = 600 detik

Cara jawabnya:

3600 detik + 600 detik = 4200 detik.

Sedikit Tambahan

4200 detik = 70 menit

16. asal usul kutoarjo , jawa tengah


Menurut sejarah, Kutoarjo pada masa penjajahan Hindia Belanda merupakan ibukota kabupaten Semawung. Pada tahun 1934 wilayahnya kemudian digabung dengan Kabupaten Purworejo. Pola pembangunan perkotaan di Kutoarjo hampir sama dengan Purworejo karena pernah menjadi pusat pemerintahan. Setelah bergabung dengan Kabupaten Purworejo posisi Kutoarjo bukan lagi sebagai ibukota kabupaen namun daerah setingkat diatas kecamatan atau dikenal dengan Kawedanan. setelah indonesia merdeka dan penetapan Jawa Tengahsebagai provinsi pada tahun 1950 kedudukan Kutoarjo hanyalah kota kecamatan. Pola pembangunan dan penataan kota yang khas dapat dilihat dari adanya Alun-alun, Pendapa, Masjid dan gedung-gedung pemerintahan yang berdekatan.

Kecamatan Kutoarjo memiliki kelebihan dibandingkan dengan kecamatan-kecamatan lain di Kabupaten Purworejo. Salah satu kelebihan yang dimiliki yaitu merupakan kota kecamatan paling ramai selain kota Kabupaten Purworejo. Kutoarjo dilewati langsung oleh Jalan Nasional Rute 3 dan jalur kerete api lintas selatan sehingga banyak pengguna kendaraan umum yang turun di Kutoarjo sebelum melanjutkan perjalanan ke kampung. Selain itu, Stasiun kereta apiterbesar di Kabupaten Purworejo berada di Kutoarjo. Stasiun Kutoarjo merupakan stasiun besar tipe B yang terletak di Kelurahan Semawung Daleman, Kutoarjo, Purworejo. Stasiun ini melayani naik turun penumpang baik kelas Ekonomi, Ekonomi Plus, Bisnis, Eksekutif dan Komuter, hampir semua kereta api penumpang lintas selatan Jawa berhenti di stasiun ini. Terdapat pula Perusahaan otobus (PO) Sumber Alam di Kutoarjo.

20. kereta api Prameks berangkat dari Jogja menuju Solo dengan jarak tempuh 64 km dari arah rute dan jalan yang sama kereta api Gajayana berangkat dari Solo menuju Jogja kecepatan kereta api Prameks dan Gajayana berturut adalah 70 km per jam dan 80 km per jam Jika kereta api berangkat pukul 08.00 Dan Api Gajayana pukul 08.12 maka kedua kereta api itu akan berpapasan pada pukul???pakai caranya​a.08.20b.08.30c.08.32d.08.42


Diketahui :

Jarak = 64 km

Waktu berangkat :

a. Prameks = 08.00 WIB

b. Gajayana = 08.12 WIB

Kecepatan :

a. Prameks = 70 km/jam

b. Gajayana = 80 km/jam

Ditanya : Waktu berpapasan?

Jawab :

Selisih waktu berangkat

[tex]08.12 - 08.00 = 12 \: menit[/tex]

Jarak yang sudah ditempuh Prameks

[tex]70 \times \frac{12}{60} = 70 \times \frac{1}{5} \\ \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: = \frac{70}{5} \\ \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: = 14 \: km[/tex]

Selisih jarak antara Prameks dan Gajayana

[tex]64 - 14 = 50 \: km[/tex]

Waktu berpapasan

[tex]waktu = \frac{jarak}{kecepatan} \\ \: \: \: \: \: \: \: \: \: \: \: \: \: = \frac{50}{(70 + 80)} \\ \: \: \: \: \: \: \: \: \: \: \: \: \: = \frac{50}{150} \\ \: \: \: \: \: \: \: \: \: \: \: \: \: = \frac{5}{15} \\ \: \: \: \: \: \: \: \: \: \: \: \: \: = \frac{1}{3} \: jam \\ \: \: \: \: \: \: \: \: \: \: \: \: \: = \frac{1}{3} \times 60 \\ \: \: \: \: \: \: \: \: \: \: \: \: \: = \frac{60}{3} \\ \: \: \: \: \: \: \: \: \: \: \: \: \: = 20 \: menit[/tex]

[tex]08.12 + 20 \: menit = 08.32[/tex]

Jadi kedua kereta tersebut berpapasan pada pukul 08.32 (C.)

21. kereta api Prameks berangkat dari Jogja menuju Solo dengan jarak tempuh 64 km dari arah rute dan jalan yang sama kereta api Gajayana berangkat dari Solo menuju Jogja kecepatan kereta api Prameks dan Gajayana berturut adalah 70 km per jam dan 80 km per jam Jika kereta api berangkat pukul 08.00 Dan Api Gajayana pukul 08.12 maka kedua kereta api itu akan berpapasan pada pukul???​


Kereta api Prameks berangkat dari Jogja menuju Solo dengan jarak tempuh 64 km, dari arah rute dan jalan yang sama kereta api Gajayana berangkat dari Solo menuju Jogja. Kecepatan kereta api Prameks dan Gajayana berturut adalah 70 km/jam dan 80 km/jam. Jika kereta api Prameks berangkat pukul 08.00 dan kereta api Gajayana pukul 08.12 maka kedua kereta api itu akan berpapasan pada pukul 08.32.

Penjelasan dengan langkah-langkah:

Kecepatan adalah jarak yang ditempuh suatu benda dalam setiap satuan waktu. Satuan untuk kecepatan yang sering digunakan adalah m/s ( meter per detik ) atau km/jam ( kilometer per jam ). Untuk menghitung kecepatan digunakan rumus:

kecepatan = jarak : waktu

Soal yang berhubungan dengan kecepatan adalah waktu menyusul dan berpapasan. Pada soal diatas ditanyakan waktu berpapasan dari kereta api Prameks dan Gajayana. Untuk menghitung waktu berpapasan digunakan rumus:

Waktu berpapasan = Jarak total : ( kecepatan I + kecepatan II )

Pada soal diatas diketahui jarak total dari Yogyakarta ke Solo adalah 64 km dan kereta api Prameks berangkat 12 menit lebih awal dari kereta Gajayana. Jarak yang digunakan untuk menghitung waktu berpapasan adalah jarak setelah kereta api Prameks berjalan selama 12 menit. Jarak kereta setelah kereta api Prameks berjalan selama 12 menit adalah:

Jarak = 64 km - ( 12 menit x kecepatan Prameks)

Jarak = 64 km - ( 12 menit x 70 km/jam )

Jarak = 64 km - ( 0,2 jam x 70 km/jam )

Jarak = 64 km - 14 km

Jarak = 50 km

Setelah diketahui jaraknya dapat dihitung waktu hingga kedua kereta berpapasan. Kecepatan kereta api Prameks adalah 70 km/jam dan kecepatan kereta api Gajayana adalah 80 km/jam.

Waktu berpapasan = 50 km : ( 70 km/jam + 80 km/jam)

Waktu berpapasan = 50 km : 150 km/jam

Waktu berpapasan = 0,333 jam

Waktu berpapasan = 20 menit

Kedua kereta akan berpapasan 20 menit setelah kereta api Gajayana berangkat. Jadi kereta akan berpapasan pada pukul 08.32.

26. Jarak kota Yogya - Kutoarjo 80 km. Pak Doni mengendarai motor berangkat pukul 07.30 dari Yogya menuju Kutoarjo. Kecepatan Pak Doni rata-rata 60 km/jam. Pak David 30 menit sebelumnya berangkat dari Kutoarjo melalui jalan yang sama. Kecepatan rata-rata Pak David 40 km/jam. Pak David dan Pak Doni berpapasan pada jarak ... dari Kutoarjo.


Jawaban:

44 km dari kutoarjo

Penjelasan dengan langkah-langkah:

Jarak = 80 kmDavid (K1) = 40 km/jam , 07:00 (30 menit sebelumnya)Doni (K2) = 60 km/jam , 07:30

Selisih jarak

= K1 x Perbedaan waktu dalam jam

= 40 km/jam x (30/60) jam

= 40 km/jam x 1/2

= 20 km

Waktu papasan

= (Jarak - selisih jarak)/K1 + K2

= (80-20) km / (40 + 60) km/jam

= 60 km / 100.km/jam

= 0,6 jam

Jarak papasan

= K1 x waktu papasan

= 40 km/jam x 0,6 jam

= 24 km

jadi mereka berpapasan pada jarak 24 km + 20 km = 44 km dari kutoarjo
