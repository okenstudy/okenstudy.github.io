17. Cara setting kartu smartfren di xiaomi redmi 4A?


Langkah Pertama

1. Masukkan kartu smartfren kmu ke HP (Slot SIM 1)
2. Masuk ke setting
3. Pilih SIM cards & mobile network
4. Pilih SIM 1 (Smartfren)
5. Pilih Acces Point Names
6. Pilih logo + New APN
7. Ketikkan: 
Nama: Smartfren
APN: Smartfren4G
Proxy: (jangan diedit)
Username: smartfren
Password: smartfren
Bawah2nya password jangan di edit
8. Selesai langkah pertama

Langkah Kedua
1. Pilih Phone
2. ketik: *#*#4636#*#*
3. Pilih Phone information1
4. Pada set preferred network type, pilih: LTE Only
5. Pencet Turn off radio (diamkan 10 detik)
6. Pencet Turn on Radio
7. Selesai

18. Hp saya kalau dimasukkan kartu memori/SD card langsung eror, kenapa?


Mapel : ti
Kelas : 12
Materi : troubleshooting
Kode soal : 11
Kode kategorisasi : 12.11

Pembahasan :
Kartu SDmu ketika dimasukkan kedalam ponsel akan error

Hal ini agak rumit untuk dijelaskan karena ada banyak penyebabnya. Bisa saja itu karena :
1. Virus
2. Karena kesalahan pabrik atau pengguna
3. Barang palsu :(
4. Kuningan kotor

Untuk mengatasi hal ini kamu mungkin perlu melakukan hal-hal dibawah ini :
1. Coba untuk mengecek kartu yang bersangkutan dengan antivirus
2. Jika kartu tersebut belum sampai 12 bulan dan kamu tidak melakukan apa-apa dengan kartumu, coba minta garansi. Jika kamu melakukan hal yang tidak biasa dengan kartumu, itu adalah kesalahan pengguna dan kesalahan ini agak sulit untuk digaransikan
3. Barang palsu adalah penyebab utama dari kerusakan kartumu. Perlu diketahui bahwa barang palsu dapat memengaruhi kualitas menyimpan data dan keamanan. Wajar saja jika kartu SD palsu, file akan korup atau rusak. Jika hal ini terjadi, coba untuk mengganti kartunya dengan yang asli atau coba format jika bisa (dengan komputer)
4. Bersihkan kuningan SD card dengan penghapus putih. Biasanya ini akan memperbaiki kesalahan dalam membaca data
