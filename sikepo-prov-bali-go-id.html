28. Pada Tahun 2021 kelahiran bayi di prov .lampung 80.000 jiwa , sedangkan jumlah penduduk prov .lampung 2.000.000 hitunglah CBR prov .Lampung ?​


[tex]\purple{\boxed{\tt{Jawaban}}}[/tex]

400 jiwa

Penjelasan :

Diketahui :Jumlah Kelahiran dalam satu tahun ( B ) = 80.000 jiwaJumlah seluruh penduduk pada pertengahan tahun ( P ) = 2.000.000 jiwa

Ditanya :

Crude Birth Rate ( CBR ) = ???

Penyelesaian :

CBR = B / P × 1.000

= 80.000 / 200.000 × 1.000

= 80.000 / 200

= 400 jiwa

Jadi, angka kelahiran kasar ( CBR ) penduduk di provinsi Lampung pada tahun 2021 adalah 400 jiwa.

Note :

1.000 = jumlah bayi yang lahir setiap 1.000 orang dari total penduduk di wilayah tersebut ( Konstanta )
