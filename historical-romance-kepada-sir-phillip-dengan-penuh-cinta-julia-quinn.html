8. Uang julia 3/5 kali uang Romi, jika romi memberikana uang Rp. 168.000 kepada julia, maka uang romi menjadi 7/9 uang julia, uang julia mula - mula adalah....


Jawab:

Uang Julia (J) uang Romi (R)

(i) J : R = 3 : 5

R = 5/3 J

(ii) (J+168.000) : (R-168.000) = 7 : 9

maka uang J awal ...?

Penjelasan dengan langkah-langkah:

masukan persaaan (i) ke (ii)

(J + 168.000) : (5/3 J - 168.000) = 7 : 9

9(J+168.000) = 7 (5/3 j - 168.000)

9J + 1.512.000 = 35/3 j - 1.176.000

9J - 35/3 J = - 1.176.000 - 1.512.000

-8/3 J = - 2.688.000

J = 2.688.000 x 3/8

J =  8.064.000/8

J = 1.008.000

uang julia mula-mula Rp.1.008.000


9. Perbandingan banyak uang rini dan Julia 2 : 5 sedangkan uang rini dan julia 147.000 selisih uang rini dan julia adalah


Pembahasan

Jumlah Perbandingan
= 2 + 5
= 7

◼ Jumlah Uang Rini

= 147.000 x 2/7
= 21.000 x 2
= Rp. 42.000

◼ Jumlah Uang Julia

= 147.000 x 5/7
= 21.000 x 5
= Rp. 105.000

Sehingga, Selisih
= 105.000 - 42.000
= Rp. 63.000.-
