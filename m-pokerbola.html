20. (m⁻² + m⁻¹ + m°) : (m⁻¹ + m° + m) = ?​


Materi : Bentuk Akar dan Pangkat

( 1/m² + 1/m + 1 ) : ( 1/m + 1 + m )

= ( 1/m² + m/m² + m²/m² ) : ( 1/m + m/m + m²/m )

= ( [ m² + m + 1 ]/m² ) : ( [ m² + m + 1 ]/m )

= ( m² + m + 1 )/m² × m/( m² + m + 1 )

= 1/m² × m/1

= m/m²

= 1/m

23. m⁻² + m⁻¹ + m°) : (m⁻¹ + m° + m) = ?​


Hasil dari m⁻² + m⁻¹ + m°) : (m⁻¹ + m° + m) adalah [tex]\frac{3}{m} [/tex]

Pendahuluan

Bilangan berpangkat merupakan bilangan yang akan menghasilkan hasil dari perkalian berulang dengan syarat memiliki bilangan yang sama.

Jadi bilangan eksponen bisa disebut juga sebagai bentuk dari sebuah bilangan yang di kalikan yaitu dengan bilangan yang sama dan dilakukan berulang ulang, eksponen di sebut dengan pangkat yang menunjukan nilai dari derajat kepangkatan.

Bilangan berpangkat adalah perkalian angka tersebut dengan angka yang sama sebanyak pangkat

a³ = a x a x a

Akar pangkat 2 adalah hasil dari suatu pangkat pecahan yaitu 1/2 √a = a^½

Akar pangkat 3 adalah pangkat ⅓ dari suatu bilangan ³√a = a^⅓

Sifat Pada Bilangan Berpangkat :

Jika pembagian maka pangkat dikurang. [tex] a^m \div a^n = a^{(m-n)} [/tex]Jika dikali maka pangkat ditambah. [tex] a^m \times a^n = a^{(m+n)} [/tex][tex] (a^m)^n = a^{m\times n} [/tex][tex] (ab)^n =a^nb^n [/tex][tex] \sqrt[n]{ {a}^{m} } = {a}^{ \frac{m}{n} } [/tex][tex] (\frac{a}{b})^n = \frac{a^n}{b^n} [/tex][tex] a^{-n} = \frac{1}{a^n} [/tex][tex] a^0 = 1 [/tex]

Untuk mengerjakan soal di atas kita simak penjelasan di bawah ini :

PembahasanDiketahui:

m⁻² + m⁻¹ + m°) : (m⁻¹ + m° + m) = ?

Ditanya:Hasilnya ?

Jawab :

[tex] = ({m}^{ - 2} + {m}^{ - 1} + {m}^{0}) \div ( {m}^{ - 1} + {m}^{0} + m)[/tex]

[tex] = ({m}^{ - 2 - ( - 1)} + {m}^{ - 1 - 0} + {m}^{0 - 1})[/tex]

[tex] = ({m}^{ - 1} + {m}^{ - 1} + {m}^{ - 1})[/tex]

[tex] = 3 {m}^{ - 1} [/tex]

[tex] = \frac{3}{m} [/tex]

Kesimpulan

Hasil dari m⁻² + m⁻¹ + m°) : (m⁻¹ + m° + m) adalah [tex]\frac{3}{m} [/tex]

1.MAJAS PERBANDINGAN

Majas perbandingan adalah majas yang membandingkan atau menyandingkan antara satu objek dengan objek lainnya. Ada pun majas yang termasuk ke dalam majas perbandingan, antara lain alegori, metafora, metonimia, litotes, hiperbola, pars pro toto, totem pro parte, dan eufimisme.

2.MAJAS METAFORA

Majas ini merupakan majas yang memakai analogi atau perumpamaan terhadap dua hal yang berbeda.

Contoh: Anak itu dikenal sebagai kutu buku di kelasnya.

3.MAJAS HOPERBOLA

Majas Hiperbola adalah majas yang digunakan untuk melebih-lebihkan sesuatu. Majas ini bertujuan untuk menekan, memperhebat, maupun memberikan kesan yang berlebihan. Contoh majas hiperbola: -Harga-harga sudah meroket.

4.MAJAS HIPERBOLA

Majas yang merupakan ungkapan yang berlebihan dan tidak masuk akal.

Contoh: Dentuman itu menggelegar membelah angkasa.
