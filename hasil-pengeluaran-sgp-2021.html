3. Berapa hasil dari 2021 log 7 / 2021 log 343?


Prof Brainly Master :

²⁰²¹ log 7/ ²⁰²¹ log 343

= ⅓ ²⁰²¹ log 7/ ²⁰²¹ log 7

= ⅓

[tex]\boxed{\boxed{\frac{^{2021} log(7)}{^{2021} log(343)} = \frac{1}{3}}} \\[/tex]

Pembahasan

Logaritma adalah operasi kebalikan atau invers dari eksponen (perpangkatan).

Rumus dasar logaritma

[tex]\boxed{^{a} log \: b = x \: \Rightarrow \: {a}^{x} = b} \\ \\ a \: \: \text{adalah basis atau bilangan pokok logaritma} \: . \\ \\ a > 0 \: \: \text{dan} \: \: a \neq 1 \\ \\ [/tex]

Sifat-sifat logaritma

[tex]\boxed{^{a} log(b) + \: ^{a} log(c) = \: ^{a} log(b \times c)} \\ \\ \boxed{^{a} log(b) - \: ^{a} log(c) = \: ^{a} log \left ( \frac{b}{c} \right )} \\ \\ \boxed{^{a} log(b) = \: \frac{1}{^{b} log(a)}} \\ \\ \boxed{^{a} log(b) = \: \frac{ log(b) }{ log(a) }} \\ \\ \boxed{^{a} log(b) \cdot \: ^{b} log(c) \cdot \: ^{c} log(a) = 1} \\ \\ \boxed{\frac{^{c} log(a)}{^{c} log(b)} = \: ^{b} log(a) } \\ \\ \boxed{\Large{{( {a}^{m} )}^{ ^{ {a}^{n} } log(b) }} = {b}^{ \frac{m}{n} }} \\ \\ [/tex]

Diketahui :

[tex]\frac{^{2021} log(7)}{^{2021} log(343)} \\ \\ [/tex]

Ditanya :

[tex]\text{Hasil dari} \: \: \frac{^{2021} log(7)}{^{2021} log(343)} \\ \\ [/tex]

Jawab :

[tex] \: \: \: \: \: \frac{^{2021} log(7)}{^{2021} log(343)} \\ \\ = \frac{ \left ( \frac{ log(7) }{ log(2021) } \right ) }{ \left ( \frac{ log(343) }{ log(2021) } \right ) } \\ \\ = \frac{ log(7) }{ log(2021) } \times \frac{ log(2021)}{ log(343) } \\ \\ = \frac{ log(7) }{ log(343) } \\ \\ = \frac{ log(7) }{ log( {7}^{3} ) } \\ \\ = \frac{ log(7) }{3 \: log(7) } \\ \\ = \frac{1}{3} \\ \\ [/tex]

Kesimpulan :

[tex]\boxed{\boxed{\frac{^{2021} log(7)}{^{2021} log(343)} = \frac{1}{3}}} \\ \\ [/tex]

9. Tentukan hasil dari :1² - 2² + 3² - 4² + 5² - 6² + ...... + 2021² - 2022²​


Jumlah n suku pertama deret aritmatika :

Sn = n/2(a + Un)

Keterangan :

Sn : Jumlah n suku pertama

n : banyaknya suku

a : suku pertama

Un : suku ke - n

Jumlah deret 1² - 2² + 3² - 4² + 5² - 6² + ...... + 2021² - 2022²​ dapat ditentukan berdasarkan pola berikut.

1² − 2² = (1 + 2)(1 - 2) = (3)(-1) = -3

3² − 4² = (3 + 4)(3 - 4) = (7)(-1) = -7

5² − 6² = (5 + 6)(5 - 6) = (11)(-1) = -11

...

2019² − 2020² = (2019 + 2020)(2019 - 2020) = (4039)(-1) = -4039

1² − 2² + 3² − 4² + ⋯ + 2019² − 2020² merupakan deret aritmatika dengan:

a = -3

Un = -4039

n = 2020/2 = 1010

Jumlahnya yaitu:

Sn = n/2(a + Un)

S1010 = 1010/2(-3 + (-4039))

S1010 = 505(-4042)

Sehingga untuk 1² − 2² + 3² − 4² + ⋯ + 2019² − 2020² + 2021² yaitu:

1² − 2² + 3² − 4² + ⋯ + 2019² − 2020² + 2021²

= 505(-4042) + 2021²

= -20412210 + 4084441

= 2.043.231

Jadi hasil dari 1² − 2² + 3² − 4² + ⋯ + 2019² − 2020² + 2021² adalah 2.043.231.

21. Pengeluaran rutin 152.789,00pengeluaran pembangunan 99.988,50penerimaan rutin 200.920,00penerimaan pembangunan 50,860,00berdasarkan data dioatas posisi apbn 2021 adalah….​


Jawaban:

Jawaban : E

Pembahasan :

Untuk mengetahui bagaimana posisi APBN, kita dapat mencarinya dengan cara penerimaan negara dikurangi dengan belanja negara.

Pendapatan negara = penerimaan rutin + penerimaan pembangunan

= Rp200.920,00 + Rp50.860,00

= Rp251.780,00

Pengeluaran negara = pengeluaran rutin + pengeluaran pembangunan

= Rp152.789,00 + Rp99.988,50

= Rp252.777,50

Sehingga, posisi APBN = Pendapatan Negara – Pengeluaran Negara

= Rp251.780,00 – Rp252.777,50

= - Rp997,50

Karena pendapatan negara lebih kecil dari pengeluaran, sehingga posisi APBN bernilai negative, nilai negative tersebut mengindikasikan posisi APBN mengalami defisit
