Yang menjaga fase cair LPG, adalah tekanan di dalam tabung. Pada tekanan lingkungan yang normal (di luar tabung), LPG berada dalam bentuk gas. Sehingga kalau LPG dikeluarkan dari tabung entah dengan cara apapun, misalnya karena bocor, LPG akan langsung berubah fase menjadi gas.

26. apakah bahan dari cairan korek api gas?


Bahan dari cairan korek api gas adalah naphta dan butana

Pembahasan:

Korek api adalah alat untuk menyalakan api. Korek api padat mengandung fosfor, sementara korek api gas mengandung cairan hidrokarbon seperti naphta dan butana.

Naphtha (atau nafta) adalah campuran hidrokarbon cair yang mudah terbakar. Nafta ringan adalah fraksi minyak bumi yang mendidih antara 30 ° C dan 90 ° C dan terdiri dari molekul dengan 5-6 atom karbon. Nafta berat mendidih antara 90 ° C dan 200 ° C dan terdiri dari molekul dengan 6-12 atom karbon.

Butana (C₄H₁₀) adalah senyawa hidrokarbon dari kelompok alkana, yaitu senyawa dengan rantai yang lurus dan tidak memiliki ikatan rangkap. Buatan memiliki 4 atom karbon. Butana memiliki sifat mudah terbakar, seperti hidrokarbon lainnya.
