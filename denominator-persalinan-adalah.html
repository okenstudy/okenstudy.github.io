11. The numerator of a fraction is 4 less than the denominator. If numerator is doubled and the denominator is subtracted by 2, the sum of the original fraction and the new one is 3. Find the original fraction.


The original fraction is: [tex] \frac{ - 3}{1} [/tex]

Explanation:

Based on the text we know that the numerator of a fraction is 4 less than the denominator.

Problem Solving:

For instance:

The denominator is equal to "a", so the numerator is "a - 4".

Meanwhile for the new fraction it's written:

The new denominator is subtracted by 2

=> a - 2

And the new numerator is doubled

=> 2(a - 4)

=> 2a - 8

The sum of the original fraction and the new one is 3, it's written:

=> [tex] \frac{a - 4}{a} + \frac{2a - 8}{a - 2 } = 3[/tex]

=> [tex] \frac{(a - 4)(a - 2) + (2a - 8)a}{a(a - 2)} = 3[/tex]

=> [tex] \frac{( {a}^{2} - 6a + 8) + 2 {a}^{2} - 8a }{ {a}^{2} - 2a } = 3[/tex]

=> [tex] \frac{ 3 {a}^{2} - 14a + 8 }{ {a}^{2} - 2a } = 3[/tex]

=> 3a² - 14a + 8 = 3(a² - 2a)

=> 3a² - 14a + 8 = 3a² - 6a

=> 3a² - 3a² - 14a + 6a = -8

=> -8a = -8

=> a = 1

The original denominator:

=> a = 1

The original numerator:

=> a - 4

=> 1 - 4 = -3

So, the original fraction is [tex] \frac{ - 3}{1} [/tex]

18. Pertukaran barang dengan barang dengan menggunakan alat di sebut sebagai denominator of value di sebut dengan?


Barter adalah kegiatan tukar-menukar barang yang terjadi tanpa perantaraan uang.
