1. Sebuah museum memberlakukan tiket masuk untuk dewasa dan anak-anak. Pada suatu hari terjual tiket masuk sebanyak 884 lembar. Perbandingan Antara banyak tiket masuk pengunjung dewasa dan tiket masuk anak-anak adalah 1 : 3. Jika x menyatakan banyak tiket dewasa yang terjual dan y menyatakan banyak tiket masuk anak-anak yang terjual, persamaan matriks dari permasalahan tersebut adalah.


Persamaan matriks dari permasalahan tersebut adalah:

[tex]\left[\begin{array}{cc}1&1\\3&-1\end{array}\right] \left[\begin{array}{c}x\\y\end{array}\right]=\left[\begin{array}{c}884\\0\end{array}\right][/tex]

Apabila dilakukan perhitungan maka akan diperoleh nilai x (tiket untuk dewasa) berjumlah 221, sedangkan nilai y (tiket untuk anak-anak) adalah 663 tiket.

Penjelasan dengan langkah-langkah

Matriks merupakan suatu bentuk penyusunan bilangan atau persamaan dalam bentuk baris dan kolom. Apabila diketahui dua persamaan linier:

ax + by = c
dx + ey = f
Di mana a, b, c, d, e, dan f adalah suatu bilangan.

Maka bentuk persamaan matriksnya adalah:

[tex]\left[\begin{array}{cc}a&b\\d&e\end{array}\right] \left[\begin{array}{c}x\\y\end{array}\right]=\left[\begin{array}{c}c\\f\end{array}\right][/tex]

Diketahui:

Jumlah tiket masuk untuk dewasa = x.Jumlah tiket masuk untuk anak-anak = y.Jumalh tiket masuk total = 884.Perbandingan tiket masuk dewasa : tiket masuk anak- anak sama dengan 1 : 3.

Ditanyakan:

Persamaan matriks = ?

Penyelesaian:

Persamaan pertama:

Dilihat dari jumlah tiket total:
x + y = 884 ...(1)Dilihat dari perbandingan:
x : y = 1 : 3
3x = y
3x - y = 0 ...(2)Maka nilai:
a = 1
b = 1
c = 88
d = 3
e = -1
f = 0

Bentuk matriks dari soal:

[tex]\left[\begin{array}{cc}1&1\\3&-1\end{array}\right] \left[\begin{array}{c}x\\y\end{array}\right]=\left[\begin{array}{c}884\\0\end{array}\right][/tex]

asumsikan :

tiket balita = a

tiket dewasa = b

maka :

a + b = 540 (pers. 1)

2000a + 3000b = 1.260.000 (pers. 2)

Substitusi pers. 1 menjadi :

a = 540 - b (pers. 3)

2000 (540 - b) + 3000b = 1.260.000

1.080.000 - 2000b + 3000b = 1.260.000

-2000b + 3000b = 1.260.000 - 1.080.000

1000b = 180.000

b = 180.000/1000

b = 180

Substitusi nilai b ke pers. 1 untuk mencari nilai a

a + b = 540

a + 180 = 540

a = 540 - 180

a = 360

maka tiket balita yang dijual sebanyak 360 tiket
