b. Berapakah rata-rata pertumbuhan ekonomi dari  tahun 2015-2018 jawaban : 5,12%

Pembahasan

Laju pertumbuhan ekonomi Tahun 2016 sampai dengan 2018 adalah

Laju pertumbuhan tahun 2016

=  {PDB 2016 - PDB 2015} / {PDB 2016}  × 100%

=  (2.658.475,0 M - 2.543.545,1 M  } / {2.658.475,0 M}  × 100%

= 4,32%

Laju pertumbuhan tahun 2017

=  {PDB 2017 - PDB 2016} / {PDB 2017}  × 100%

=  (2.764.124,2 - 2.658.475,0 M  } / { 2.764.124,2 M}  × 100%

= 3,82%

Laju pertumbuhan tahun 2018

=  {PDB 2018 - PDB 2017} / {PDB 2018}  × 100%

=  ( 2.980.090.0 - 2.764.124,2 M  } / { 2.980.090.0 M}  × 100%

= 7,24%

Maka rata-rata pertumbuhan ekonomi dari tahun 2015 sampai dengan 2018 adalah :

4,32 + 3,82 + 7,24 / 3 = 5,12%
