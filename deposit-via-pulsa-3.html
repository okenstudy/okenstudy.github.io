7. apa yang dimaksud dengan primary deposit dan loan deposit


Primary deposit dan loan deposit adalah cara-cara untuk menciptakan uang giral.

Primary deposit adalah cara menciptakan uang giral dengan cara nasabah menyetorkan uang kartalnya ke bank untuk dicatat dalam rekening koran miliknya sebagai uang giral.

Loan deposit adalah cara menciptakan uang giral yang terjadi ketika seorang nasabah meminjam uang ke bank, namun uang tsb tidak ditarik menjadi uang kartal, namun tetap dibiarkan didalam rekening korang miliknya.

11. apa arti loan deposit dan primary deposit ...??


Penyetoran uang tunai kepada bank dan dicatat dalam rekening koran atas nama penyetor, penyetor menerima buku cek dan buku giro bilyet. Uang tersebut sewaktu-waktu dapat diambil atau penyetor menerima pembayaran utang dari debitur melalui bank. Penerimaan piutang itu oleh bank dibukukan dalam rekening koran orang yang bersangkutan. 12. Jumlah uang via dan amel Rp. 28. 000,00 perbandingan kelereng via dan amel 3 : 4. Berapakah uang via ?​


Jumlah uang Via adalah Rp 12.000,00. Untuk menyelesaikan soal ini kita gunakan formula:

Uang Via = rasio uang via / (rasio jumlah uang Via dan Amel) x jumlah uang Via dan Amel.

Penjelasan dengan langkah-langkah:

Diketahui:

Jumlah uang via dan amel Rp. 28.000,00

Perbandingan kelereng Via dan Amel 3 : 4.

Ditanya:

Berapakah uang via ?

Jawab:

Untuk menyelesaikan soal ini kita gunakan formula:

Uang Via = rasio uang via / (rasio jumlah uang Via dan Amel) x jumlah uang Via dan Amel.

Uang Via = 3/(3 + 4) x Rp. 28.000,00

               = 3/7 x Rp. 28.000,00

               = 3 x Rp. 4.000,00

               = Rp 12.000,00

Jumlah uang Via adalah Rp 12.000,00.
