16. Carilah objek-objek baru yang tidak ada di aplikasi kalkulator, dan tulislah dalam lembar kerja berikut!​


Berikut ini adalah objek-objek baru yang tidak ada di aplikasi kalkulator beserta dengan kegunaannya:

ln (Logaritma alami)

Untuk mendefinisikan untuk semua bilangan real (riil) positif x dan dapat juga didefinisikan untuk bilangan kompleks yang bukan 0.

' (Menit)

Untuk mengkonversi sudut ke satuan menit.

' ' (Detik)

Untuk mengkonversi sudut ke satuan detik.

≥ (lebih dari atau sama dengan) dan ≤ (kurang dari atau sama dengan)

Untuk membandingkan angka dan ekspresi.

Pembahasan

Kalkulator adalah alat yang digunakan untuk menghitung, banyak orang yang kesulitan menghitung tanpa kalkulator dikarenakan kurangnya belajar dan hafalan tentang hitung-hitungan, seperti penambahan, pengurangan, pengalian dan pembagian angkaangka. Untuk itu kalkulator adalah alat yang bermanfaat bagi seseorang yang kesulitan didalam menghitung terutama mata pelajaran matematika dan juga pekerjaan sehari-hari. Untuk itu mari kita simak ulasan tentang kalkulator.

Pengertian kalkulator secara singkat ialah alat bantu untuk menghitung. Kalkulator merupakan alat hitung elektronika yang jauh lebih sederhana dibandingkan dengan komputer, dan dikalangan masyarakat sudah banyak yang menggunakannya sebagai alat bantu hitung yang praktis dan cepat. Dan saat ini sudah banyak beredar kalkulator dengan bermacam-macam merek dan tipe, yang biasanya mempunyai cara pengoperasian yang berbeda-beda, tetapi pada dasarnya hampir sama. Penggunaan kalkulator perlu dilakukan karena kalkulator merupakan alat yang mampu membantu memudahkan perhitungan. Pengoperasian bilangan-bilangan besar dapat dilakukan dengan mudah kalau kita menggunakan kalkulator.

20. Layaknya kakulator,Microsoft Excel dapat digunakan untuk penjumlahan,pengurangan,perkalian,dan pembagian.uraikan apa yang menjadi kelebihan aplikasi dengan kalkulator sebagai alat mengelola angka​


BISMILLAH

Berikut beberapa kelebihan Microsoft Excel dibandingkan dengan kalkulator sebagai alat untuk mengelola angka:

Kapasitas Data yg Lebih Besar: Microsoft Excel memungkinkan pengguna untuk mengelola jumlah data yg jauh lebih besar dibandingkan dengan kalkulator biasa.

Lebih Dari Sekedar Operasi Matematika: Selain dapat melakukan operasi matematika seperti penjumlahan, pengurangan, perkalian, dan pembagian, Microsoft Excel juga memiliki fitur-fitur yg lebih kompleks seperti fungsi statistik, grafik, dan pivot table.

Kemampuan Visualisasi Data: Microsoft Excel memungkinkan pengguna untuk menampilkan data dalam bentuk grafik atau tabel, sehingga memudahkan dalam memahami dan mempresentasikan data.

Kemampuan Automatisasi: Microsoft Excel memiliki banyak formula dan fungsi yg memungkinkan pengguna untuk mengautomasi proses-proses seperti menghitung total, rata-rata, dan deviasi standar.

Kemampuan Collaboration: Microsoft Excel memiliki fitur yg memungkinkan beberapa orang untuk bekerja pada file yang sama secara bersamaan dan menyimpan versi terbaru dari file tersebut.

Dapat Disimpan dan Diteruskan: Data yg dikelola dengan Microsoft Excel dapat disimpan dan diteruskan dalam bentuk file yg dapat dibuka dengan mudah dan dimanfaatkan oleh banyak orang.

21. Buatlah flowchart untuk membuat aplikasi kalkulator!​


Jawaban:

Lalu dari flowhcart diatas diimplementasikan ke program menjadi

#include<stdio.h>

#include<conio.h>

int main()

{

float a, b, hasil; char op;

puts("====================================");

puts("Kalkulator Sederhana");

puts("====================================");

printf("Angka ke-1= "); scanf("%f", &a);

printf("Angka ke-2= "); scanf("%f", &b);

printf("operasi(+,-,/,*)= "); scanf("%s", &op);

if(op=='+') hasil = a + b ;

else if(op=='-') hasil = a - b ;

else if(op=='*') hasil = a * b ;

else if(op=='/') hasil = a / b ;

printf("Hasil = %.2f",hasil);

getch();

return(0);

}

KETERANGAN PROGRAM:

>stdio.h adalah library pada bahasa C yang digunakan untuk operasi input-output (stdio = Standar Input dan Output).

>conio.h berkas header pada bahasa pemrograman C yang kebanyakan digunakan pada compiler berbasis MS-DOS untuk membuat antarmuka konsol.

>int main() adalah fungsi utama dari sebuah kode bahasa C.

>float adalah tipe data angka yang memiliki bagian desimal di akhir angka.

>char adalah tipe data yang hanya bisa diisi dengan 1 karakter saja

>Printf digunakan untuk membuat sebuah output pada tampilan

>puts berfungsi untuk membuat sebuah output pada tampilan

>if dan else if adalah statement yang digunakan untuk penyeleksian kondisi.

>getch() digunakan untuk membaca input dari keyboard dan menghentikan proses yang berjalan.

>return 0 digunakan untuk membalikkan nilai/mengakhiri program.

2.    Jika anda sudah membuka lembar baru microsoft Excel, silahkan anda memilih tombol menu Developer pada menu Ribbon, dan jika pada Ms. Excel anda belum terdapat menu Developers, anda dapat memunculkannya dengan cara membuka menu File atau dengan cara menekan logo Ms. Office di pojok kiri atas, pilih Option, dan ceklist Developer Menu.

Sehingga muncul jendela pop-up baru seperti berikut:

4.    Selanjutnya silakan memasukkan perintah InsertUserform, sehingga muncul perintah jendela pop-up userform

5.    Selanjutnya buatlah CommandButton sebanyak enam buah, dan 3 buah textbook menggunakan tools tersebut.

6.    Selanjutnya, anda dapat melakukan double klik pada setiap tombol command yang telah anda buat sebelumnya, untuk diisi kode seperti di bawa ini

Private Sub bagi_Click()

hasil = A / B

End Sub

Private Sub hapus_Click()

A = ""

B = ""

End Sub

Private Sub kali_Click()

End Sub

Private Sub pangkat_Click()

hasil = A ^ B

End Sub

Private Sub plus_Click()

hasil = Val(A) + Val(B)

End Sub
