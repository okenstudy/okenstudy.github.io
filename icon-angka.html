13. Apakah fungsi icon dari : 1.icon program 2.icon document 3.icon settings


1. icon program ialah icon yang berfungsi menampilkan program/ software bila didobel klik dengan mouse.

2. icon document ialah icon yang berfungsi menampilkan document bila didobel klik dengan mouse.

3. icon settings ialah icon yang berfungsi menampilkan jendela pensettingan tertentu saat didobel klik dengan mouse.

-icon umumnya dapat dilihat di desktop, namun icon settings paling banyak dijumpai di jendela control panel (perhatikan gambar berikut/ pada tanda-tanda panah merah)-
