26. Pada hari pertama pergelaran seni di hadiri oleh 1000 penonton.Pada hari kedua,pergelaran seni tersebut di hadiri 1050 penonton .Jika peningkatan jumlah penonton setiap hari tetap ,maka tentukan banyak penonton yang hadir pada hari ke 15！


Jawaban:

1.700 penonton yang hadir hari ke 15

Penjelasan dengan langkah-langkah:

diket : hari pertama 1.000 penonton

hari kedua 1.050 penonton

selisih h pertama dan h kedua adalah 50

( 1.000 - 1.050 = 50 )

ditanya : hari ke 15 ( U15 )

rumus Un = a + ( n - 1 ) b

jawab U15 = 1.000 + ( 15 -1 ) 50

= 1.000 + ( 14 ) 50

= 1.000 + 700

= 1.700 penonton
