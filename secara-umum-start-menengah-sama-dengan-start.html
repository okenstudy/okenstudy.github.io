13. Tugas jelaskan cara melakukan start pendek ,(bunch start) start menengah (medium start ) dan start panjang (long start ) ​


Berikut adalah cara pelaksanaan dari short start, medium start dan long start untuk cabang lari jarak pendek.

Short Start :

Kaki kanan di depan, kaki kiri di belakang. Jari-jari kaki belakang segaris dengan tumit kaki depan

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Medium Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan ujung jari

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Long Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan tumit

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

16. jelaskan perbedaan start menengah dengan start pendek!


START PENDEK (BUNCH START)

Start pendek adalah lutut kaki belakang (kaki kiri) diletakkan didepan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.


START MENENGAH (MEDIUM START)

Start menengah adalah lutut kaki belakang (kaki kiri) diletakkan sejajar dengan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.

Perbedaannya hanya terletak pada posisi lutut dengan telapak kaki.
Apabila start pendek maka posisi lutut sejajar dengan jari-jari kaki
Apabila start menengah maka posisi lutut sejajar dengan tengah telapak kaki

17. Start yang digunakan pada nomor lari jarak menengah 800 m adalah . . . . A) Start jongkok B) Start berdiri C) Start melayang D) Start berlari E) Start menengah


Start yang digunakan pada nomor lari jarak menengah 800 m adalah Start jongkok.

PEMBAHASAN

Lari jarak menengah adalah jenis cabang atletik nomor lari yang dilakukan pada jarak 800 m, 1.500 m, dan juga 3.000 m. Dalam cabang ini, ada beberapa jenis pihak yang berperan dalam perlombaan yaitu:

Starter, pihak yang mengantarkan/memberangkatkan pelari jarak menengah. Recall Starter, pihak yang mengkordinir pengecekan dan komunikasi kepada pelari. Timer, pihak yang mencatat waktu perlombaan. Pengawas lapangan, pihak yang menjadi pengawas di lapangan langsung dimana ia akan mengecek adakah pelari yang melakukan pelanggaran atau tidak. Juri kedatangan, pihak yang melihat siapa pelari yang pertama sampai di garis finish. Juri pencatat hasil, pihak yang mencatat hasil perlombaan dan menentukan siapa yang ranking 1 dst.

Mengenai start, start merupakan sikap orientatif (awalan) yang dilakukan pada olahraga cabang atletik yaitu nomor lari. Start dibedakan menjadi 3 jenis yaitu start jongkok, start berdiri, dan start melayang. Pada lari jarak menengah ada dua jenis start yang digunakan yaitu:

Start jongkok yang dikhususkan pada lari jarak menengah 800 meter.Start berdiri yang dikhususkan pada lari jarak menengah 1.500 dan 3.000 meter.

28. Gerak spesifik start pada lari jarak pendek terdiri dari tiga macam, yaitu ...A. long start, star menengah, medium startB. long start, star menengah, short startC. start panjang, star menengah, long starD. long start, short start, start pendek ​

Jawab : B. long start, star menengah, short start

Jawaban:

B. Long start, star menengah, short start.

Penjelasan:

Gerak spesifik start pada lari jarak pendek terdiri dari tiga macam yaitu long start, star menengah, dan short start. Long start adalah gerakan awal yang dilakukan dengan lari sejauh 20-30 meter sebelum garis start. Star menengah dilakukan dengan berlari sejauh 10-15 meter sebelum garis start. Sedangkan short start adalah gerakan awal yang dilakukan dengan berlari sejauh 5-10 meter sebelum garis start. Pemilihan jenis gerak start yang tepat dapat meningkatkan kecepatan dan efisiensi pada saat awal start dalam lari jarak pendek.
