3. suhu di London adalah 94°f tentukan suhu kota London dalam reamur dan Celcius ​


Jawaban:

➥ Langkah - Langkah

Derajat Reamur :

94°F = ... °R

°R = 4/9 × ( °F - 32 )

°R = 4/9 × ( 94 - 32 )

°R = 4/9 × 62

°R = 27,55°R ✅

Derajat Celsius

94°F = ... °C

°C = 5/9 × ( °F - 32 )

°C = 5/9 × ( 94 - 32 )

°C = 5/9 × 62

°C = 34,44°C ✅

L = suhu kota London
P = suhu kota Paris
K = suhu kota Kyoto
L = 4C°
P = L - 5C°
= -1C°
K + P = 12C°
K = 12C° - (-1C°)
= 12C° + 1C°
= 13C°

“di bawah suhu” berarti dikurangi suhu tersebut “di atas suhu” berarti ditambah suhu tersebut
Pembahasan
Suhu di Jakarta = 28ᵒ C Suhu di London = 30ᵒ C di bawah suhu kota Jakarta

Artinya selisih antara suhu kota London dan kota jakarta adalah 30ᵒ C dengan suhu di kota jakarta lebih besar dari suhu di kota London

Sehingga suhu di kota London adalah

= 28ᵒ C – 30ᵒ C

= –2ᵒ C
