8. arti dari yamete kudasai​


Arti dari 辞めて下さい (Yamete kudasai) adalah "Tolong hentikan".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

'Yamete kudasai' berarti "Tolong hentikan", merupakan kata kerja yang diubah bentuknya ke bentuk Te, kemudian ditambahkan kata Kudasai. Bentuk awal dari kata tersebut adalah Yameru yang artinya "Berhenti". Pembahasan lebih lanjutnya akan saya bahas per-kata, sebagai berikut.

辞めて・やめて (Yamete)

Artinya adalah "Berhenti" atau "Hentikan". Berhenti di sini maksudnya bisa berhenti melakukan sesuatu, ataupun berhenti dari sesuatu (misalnya dari pekerjaan). Kata ini merupakan bentuk Te dari kata kerja Yameru yang berarti "Berhenti". Bentuk Te sendiri bisa berfungsi untuk membuat kata perintah, membuat kata sambung, dan lainnya. Namun di sini berfungsi untuk membuat kata perintah.

下さい・ください (Kudasai)

Artinya "Mohon", "Tolong", dan lain-lain. Kata ini tidak bisa berdiri sendiri, jadi harus ada sesuatu yang diterangkannya (apa yang dimohonkannya). Kata ini kalau digabung dengan kata kerja bentuk Te, akan membuat kata tersebut menjadi kata permintaan atau permohonan yang sopan.

- - - - - - - - - -

て形 (Te-kei) / Bentuk Te

Merupakan salah satu bentuk konjugasi dalam Bahasa Jepang. Bentuk ini sangat sering digunakan dalam Bahasa Jepang. Bisa untuk kata kerja dan kata sifat, tapi kali ini saya hanya akan fokus ke bentuk Te untuk kata kerja saja.

Fungsi KK Bentuk Te

Kalau pada kata kerja, bentuk Te ini memiliki fungsi antara lain untuk :

Menghubungkan kata atau kalimat secara berurutanMembuat kata perintah atau permintaan yang sopanMenunjukkan keterangan caraMenyatakan sebab-akibat

Cara Mengubah KK ke Bentuk Te

Caranya berbeda-beda untuk tiap kelompok kata kerja, sebagai berikut.

1. Untuk KK Kelompok 1

=> Dengan mengubah akhiran :

U, Tsu, Ru menjadi TteMu, Nu, Bu menjadi NdeKu menjadi IteGu menjadi IdeSu menjadi Shιte

Contohnya seperti Kau menjadi Katte, Nomu menjadi Nonde, Aruku menjadi Aruite, Oyogu menjadi Oyoide, dan Mawasu menjadi Mawashιte.

2. Untuk KK Kelompok 2

=> Dengan mengubah akhiran :

Ru menjadi Te

Contohnya seperti Miru menjadi Mite, dan Taberu menjadi Tabete.

3. Untuk KK Kelompok 3

=> Dengan mengubah :

Suru menjadi ShιteKuru menjadi Kite

Contohnya seperti Benkyou suru menjadi Benkyou shιte, dan Modotte kuru menjadi Modotte kite.

Pembahasan

Kalimat Perintah dalam Bahasa Jepang terdiri dari beberapa kriteria sesuai penempatannya. Ada yang bentuknya sopan ada juga yang bentuknya kasar. Untuk bentuk sopan biasanya menggunakan akhiran ください (Kudasai) dan ちょうだい (Choudai). Untuk bentuk yang tergolong kasar biasanya menggunakan akhiran せ (se), ろ (ro), dan え (e).

Penggunaan akhiran ください

Secara umum kata tersebut memiliki arti Tolong, Mohon, dan sering digunakan untuk menggantikan akhiran -lah sebagai bentuk perintah serta akhiran ini bersifat sopan. Sebelum menambahkan akhiran ini kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh :

Tolong Tunggu Sebentar

Tunggu dalam Bahasa Jepang 待つ (Matsu) diubah ke bentuk -te menjadi 待って (Matte)Sebentar dalam Bahasa Jepang ちょっと (Chotto)

Sehingga polanya :

Kata Kerja bentuk -te + Kudasai

= Chotto Matte + Kudasai

= ちょっと 待って ください

Penggunaan akhiran ちょうだい

Secara umum kata tersebut memiliki fungsi yang sama dengan kudasai namun lebih ke bentuk permohonan.

. Cara menggunakan pola Choudai juga sama dengan kudasai yaitu kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh :

Mohon beri anak saya makan

Beri makan dalam Bahasa Jepang 養っ (Yashinasu) diubah ke bentuk -te menjadi 養って(Yashinate)Anak saya dalam Bahasa Jepang 私の子供 (Watashi no kodomo) ditambah partikel を (o)

Sehingga polanya :

Kata kerja bentuk -te + Coudai

= Watashi no kodomo o yashinate + Choudai

= 私の子供を養って ちょうだい

Penggunaan akhiran せ (se), ろ (ro) dan え (e)

Penggunaan akhiran -se dan -e dilakukan pada Kata Kerja 1 sedangkan -ro pada kata kerja 2. Akhiran ini berfungsi sebagai akhiran kalimat perintah yang sifatnya marah atau membentak. Untuk membuat pola kalimatnya kata kerja masu harus diubah ke bentuk se, ro, atau e. Contoh :

Bicaralah!!

Berbicara dalam Bahasa Jepang はなす(Hanasu)

Nah, berbicara termasuk Kata Kerja 1 maka harus diubah ke bentuk e atau se. Maka, ketika diubah menjadi はなせ (Hanase)

19. Quizz for youBahasa JepangArti kata ganbatte kudasai dan chotto mattenote:​Cuman Quizz


頑張って下さい (ganbatte kudasai) : semangat ya, berjuanglah, semoga beruntung.ちょっと待って (chotto matte) : tunggu sebentar.

______________

Pembahasan

Ganbatte kudasai

「頑張って下さい・がんばってください」

= Semangat ya, berjuanglah, semoga beruntung.

Untuk pemakaian frasa ini tergolong formal, juga bersifat umum. Merupakan salah satu frasa yang digunakan sebagai ungkapan penyemangat atau memberikan motivasi agar dapat bersemangat kembali.

Frasa diatas merupakan bentuk frasa dari pola kata kerja -te + 下さい (kudasai).

Kata 頑張って (ganbatte) tergolong kata kerja, yang dimana awalnya kata kerja tersebut dituliskan 頑張る (ganbaru) artinya yaitu berjuang, semangat, berusaha.

Kemudian untuk kata 下さい (kudasai) ini dapat digunakan untuk meminta atau memohon kepada orang lain, dan juga bisa digunakan sebagai bentuk kata perintah dengan kesan sopan.

Kita juga bisa menggunakan kata 頑張って (ganbatte) saja sebagai kata penyemangat kepada orang lain, namun dengan konteks digunakan dalam situasi yang informal atau akrab.

--------

Chotto matte

「ちょっと待って・ちょっとまって」

= Tunggu sebentar.

Merupakan frasa yang digunakan dalam kondisi yang informal, sehingga tidak dapat digunakan dalam nuansa formal.

Kata ちょっと (chotto) artinya yaitu sedikit, bentar/sebentar. Kemudian untuk arti 待って (matte) adalah tunggu/menunggu, merupakan golongan kata kerja konjugasi -te. Kata kerja awalan 待って (matte) adalah 待つ (matsu).

23. apa arti yamete kudasai?​


やめて下さい (yamete kudasai) artinya adalah tolong berhenti.

Pembahasan

やめて (yamete) artinya berhenti.

Ada kata selain yamete yang artinya pun juga sama, yakni やめる dibaca sebagai yameru.

下さい (kudasai) artinya seperti memohon atau meminta tolong.

Umumnya kata kudasai ini merupakan kata formal.

Sehingga ketika kata yamete maupun yamerui bertemu dengan kata lainnya yakni kudasai, artinya yang sebelumnya adalah hentikan menjadi tolong hentikan (yamete kudasai) やめて下さい .

Contoh :

1. Anisa, yamete kudasai! = Anisa, tolong hentikan!

2. Mou yamete kudasai! = Sudah, tolong berhenti!

Pembahasan:

Yamete kudasai berasal dari dua kata, yakni yamete (やめて) yang berarti hentikan, serta kudasai (うま) yang artinya tolong.

Oleh karena itu, arti yamete kudasai dalam bahasa Indonesia adalah 'tolong hentikan' atau 'berhentilah!'.

Adapun arti dari yamete kudasai punya makna yang sama dengan yameru (やめる) yang artinya berhenti atau hentikan.

Jika diartikan dengan kata kudasai, yameru kudasai (くださいめる) artinya tolong berhenti atau tolong hentikan.

25. apa arti yamate kudasai?​


やめてください (Yamete kudasai) artinya adalah "Tolong hentikan" atau "Tolong berhenti".

Pembahasan

Kata 辞めて (Yamete) merupakan bentuk Te dari kata kerja 辞める (Yameru) yang artinya "Berhenti". Fungsi bentuk Te ini salah satunya bisa untuk membuat kata perintah. Biasa juga ditambahkan ください (Kudasai) di belakangnya, agar lebih sopan. Kata 'kudasai' ini sendiri bisa berarti "Tolong", "Mohon", dsb. Kata 'kudasai' dapat digantikan dengan 'kure' untuk versi lebih kasarnya.

Berikut beberapa cara lain untuk mengatakan "Tolong hentikan" dalam Bahasa Jepang.

Pembahasan

KaIimat Perintah daIam Bahasa Jepang terdiri dari beberapa kriteria sesuai penempatannya. Ada yang bentuknya sopan dan ada juga yang bentuknya kasar. Untuk bentuk sopan biasanya menggunakan bentuk ください (Kudasai) & ちょうだい (Choudai). Untuk kata yang tergoIong kasar biasanya menggunakan bentuk せ (se), ろ (ro), dan え (e).

Penggunaan Bentuk ください

Secara umum kata tersebut memiliki arti ToIong, Mohon, dan sering digunakan untuk menggantikan akhiran -lah sebagai bentuk perintah serta akhiran ini bersifat sopan. Sebelum menambahkan akhiran ini kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh:

ToIong Bantu dia:

Membantu dalam Bahasa Jepang 助けつ (Tasuketsu). diubah ke bentuk -te menjadi 助けて (Tasukete).Dia dalam bahasa jepang 彼 (Kare) + partikel を (o).

Sehingga polanya:

S + Kata Kerja bentuk -te + Kudasai.

= Kare o + Tasukete + Kudasai.

= 彼を助けてください

Penggunaan bentuk ちょうだい

Secara umum kata tersebut memiliki fungsi yang sama dengan kudasai cuma lebih ke bentuk permohonan. Cara menggunakan pola Choudai juga sama dengan kudasai yaitu kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh:

Mohon Kerjakan PR-ku:

Kerjakan dalam Bahasa Jepang します (Shimasu). diubah ke bentuk te menjadi して (Sh|te).PR dalam Bahasa Jepang 宿題 (Shukudai) ditambah partikel を (o).

Sehingga polanya:

O + Kata kerja bentuk -te + Coudai.

= Shukudai o + sh|te + Coudai.

= 宿題をしてちょうだい

Penggunaan bentuk せ (se), ろ (ro) dan え (e).

Penggunaan bentuk -se dan -e dilakukan pada Kata Kerja 1 sedangkan -ro pada kata kerja 2. Akhiran ini berfungsi sebagai akhiran kalimat perintah yang sifatnya marah atau membentak. Untuk membuat pola kalimatnya kata kerja masu harus diubah ke bentuk se, ro, atau e. Contoh:

Pergilah!!

Pergi dalam Bahasa Jepang 行く (Iku). Nah, pergi termasuk Kata Kerja 1 maka harus diubah ke bentuk e atau se. Maka, ketika diubah menjadi 行け！(Ise!)
