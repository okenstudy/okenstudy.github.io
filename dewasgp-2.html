Rp. 270.000,-

Penjelasan dengan langkah-langkah:

Gunakan pemisalan:

orang dewasa = a

anak anak = b

2a + 2b = 140.000

a + 3b = 130.000

eliminasi:

2a + 2b = 140.000

2a + 6b = 260.000

4b = 120.000

b = 30.000

maka sudah diketahui harga 1 tiket anak anak adalah 30.000

substitusi:

a + 3b = 130.000

a + 3 (30.000) = 130.000

a + 90.000 = 130.000

a = 130.000 - 90.000

a = 40.000

maka harga 3 tiket orang dewasa dan 5 anak anak adalah:

3a + 5b

3 (40.000) + 5 (30.000)

120.000 + 150.000

=270.000,-

a) x + 4y = 550

   2x + 7y = 1000

b) x + 4y = 550 | x2

   2x + 7y = 1000 | x1

   2x + 8y = 1100

   2x + 7y = 1000

   ___________ -

   y = 100

   x + 4(100) = 550

   x + 400 = 550

   x = 150

Jadi x + y = 250 yen
