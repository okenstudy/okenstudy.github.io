Kombinasi

PHANTOM

~Kaidah Pencacahan

_____________________

Kombinasi dari kata "jocel" adalah 5 susunanKombinasi dari kata "phantom" adalah 7 susunan

• • •

Pendahuluan

Untuk rumus umum kombinasi adalah C = n! ÷ [(n - k)! × k!], Namun jika tidak ada unsur ganda dapat menggunakan rumus C = n! ÷ (n - 1)!

Keterangan:

n ➞ Total unsur/hurufk ➞ Unsur/huruf ganda

• • •

» Penyelesaian Soal

Soal Pertama

Kata "jocel" tidak memiliki unsur ganda, dari hal tersebut dapat menggunakan rumus C = n! ÷ (n - 1)!, Untuk n total unsur

Maka:

C = n! ÷ (n - 1)!

C = 5! ÷ (5 - 1)!

C = 5! ÷ 4!

C = 5 × 4!/4!

C = 5 susunan

...

Soal Kedua

Kata "phantom" tidak memiliki unsur ganda, dari hal tersebut dapat menggunakan rumus C = n! ÷ (n - 1)!, Untuk n total unsur

Maka:

C = n! ÷ (n - 1)!

C = 7! ÷ (7 - 1)!

C = 7! ÷ 6!

C = 7 × 6!/6!

C = 7 susunan
