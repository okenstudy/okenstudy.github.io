1. sebuah transformator mempunyai daya masuk 200w dan efisiensi 75%, berapakah daya yang keluar​


Jawaban:

Efisiensi transformator adalah rasio antara daya yang keluar dan daya yang masuk. Jika efisiensi transformator adalah 75%, maka 75% dari daya yang masuk digunakan untuk daya yang keluar, sisanya hilang sebagai panas.

Untuk menentukan daya yang keluar, kita dapat menggunakan rumus berikut:

Daya yang keluar = Efisiensi x Daya masuk

Daya yang keluar = 0.75 x 200W = 150W

Jadi, daya yang keluar transformator adalah 150 watt.

14. tuliskan rumus efisiensi transformator


Efisiensi Trafo dapat didefinisikan sebagai Perbandingan antara daya listrik keluaran (Pout) dengan daya listrik masukan (Pin). Efisiensi Trafo dapat dirumuskan dengan Rumus berikut ini :

ɳ = (Pout / Pin) x 100%

Dimana :

ɳ = Efisiensi Trafo
Pout : Daya listrik Keluaran (Output) atau Daya pada Kumparan Sekunder
Pin : Daya listrik Masukan (Input) atau Daya pada Kumparan Primer

Rumus-rumus turunan untuk Efisiensi Trafo lainnya :

ɳ = (Vs x Is / Vp x Ip) x 100%

atau

ɳ = (Pout / (Pout + Copper loss + Core loss) x 100%

atau

ɳ = (Pin – Losses) / (Pin) x 100%

atau

ɳ = (Ns x Is / Np x Ip) x 100%

Dimana :

ɳ : Efisiensi Trafo
Vs : Tegangan Sekunder
Vp : Tegangan Primer
Is : Arus Sekunder
Ip : Arus Primer
Ns : Lilitan sekunder
Np : Lilitan primerRumus efisiensi transformator:

[tex]efisiensi = \frac{daya \: output}{daya \: input} \times 100\% \\ \\ \\ efisiensi = \frac{tegangan \: output \times kuat \: arus \: output}{tegangan \: input \times kuat \: arus \: input} \times 100\%[/tex]

19. sebuah transformator mengubah tegangan dari 250 volt, efisiensi transformator 70%, kumparan sekunder dengan lemari pendingin 75 watt dan 100 volt. kuat arus kumparan primer adalah


Trafo dibagi 2 yaitu trafo step up dan step down. Ciri trafo step up yaitu nilai Vs > Vp. Ciri lain yaitu Ns > Np , Is < Ip. Ciri trafo step down yaitu nilai Vs < Vp. Ciri lain yaitu Ns < Np dan Is > Ip.


Pembahasan

Berdasarkan soal diatas diketahui bahwa :

sebuah transformator mengubah tegangan dari 250 volt, efisiensi transformator 70%, kumparan sekunder dengan lemari pendingin 75 watt dan 100 volt -> Vp = 250 volt, η = 70%, ps = 75 watt, Vs = 100 volt

Adapun yang ditanyakan pada soal adalah menentukan kuat arus kumparan primer (Ip).

Untuk menyelesaikan soal diatas pertama - tama mencari kuat arus sekundernya dengan menggunakan cara sebagai berikut:

Ps = Vs x Is

Is = Ps/Vs

Is = 75/100

Is = 0,75 A

Nah, setelah diketahui kuat arus sekundernya, kita tinggal mencari kuat arus primer dengan cara sebagai berikut:

Vp/Vs = Is/Ip

Vs x Is = Vp x Ip

Ip = Vs x Is/Vp

Ip = 100 x 0,75 / 250

Ip = 75/250

Ip = 0,3 A

Dengan demikian dapat disimpulkan bahwa besarnyakuat arus pad akumparan primer adalah 0,3 A.
