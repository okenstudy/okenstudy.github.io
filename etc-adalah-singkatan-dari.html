Penjelasan:

File sip.conf terletak di /etc/asterisk/sip.conf. File ini dapat dibuka dengan berbagai text editor. Seperti gedit, vim dan nano. Bisa ditulis:

<perintah_teks_editor> <path>

Di mana <perintah_teks_editor> diisi command untuk memanggil text editor dan <path> diisi jalur menuju file beserta file dan ekstensinya.

28. Apa kepanjangan dari etc, dan bagaimana cara membacanya?


Kepanjangan dari etc. adalah et cetera, cara membacanya adalah et setera.

Pembahasan

Et cetera digunakan pada akhir sebuah daftar untuk menunjukan bahwa ada poin/hal lain yang serupa. Dalam bahasa Indonesia dikenal dengan istilah dll. (dan lain-lain)

etc. dapat digunakan di tengah-tengah kalimat, maka diikuti dengan koma.

Contoh: Tennis, soccer, baseball, etc., are outdoor games.

Namun, jika kata tersebut berada di akhir sebuah kalimat kemudian diakhiri dengan tanda titik, maka tidak harus diikuti dengan koma. Tanda titiknya juga mewakili tanda akhir sebuah kalimat.

Contoh: Being outdoors, we played tennis, soccer, baseball, etc.
