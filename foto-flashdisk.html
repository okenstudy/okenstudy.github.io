13. sebuah toko komputer mempunyai persediaan flashdisk ukuran 2GB sebanyak 30buah ,flashdisk ukuran 4GB sebanyak 45 buah dan flashdisk ukuran 8GB sebanyak 27 buah,tentukan A.Perbandingan ketiga jenis flashdisk dalam bentuk paling sederhana B.Perbandingan ketiga ketiga jenis flashdisk jika telah terjual 5 flashdisk ukurn 2GB dan 7 flashdisk ukuran 8GB


Sebuah toko komputer mempunyai persediaan flashdisk ukuran 2 GB sebanyak 30 buah, flashdisk ukuran 4 GB sebanyak 45 buah dan flashdisk ukuran 8 GB sebanyak 27 buah. Perbandingan adalah bentuk sederhana dari beberapa nilai dari besaran yang sejenis untuk membandingkan nilai-nilai besaran tersebut.


Pembahasan

Diketahui

jumlah flashdisk ukuran 2 GB = 30 buah jumlah flashdisk ukuran 4 GB = 45 buah  jumlah flashdisk ukuran 8 GB = 27 buah

Ditanyakan

a. Perbandingan ketiga jenis flashdisk dalam bentuk paling sederhana  

b. Perbandingan ketiga ketiga jenis flashdisk jika telah terjual 5 flashdisk ukuran 2 GB dan 7 flashdisk ukuran 8 GB


Jawab

a. Perbandingan ketiga jenis flashdisk dalam bentuk paling sederhana

= flasdisk 2 GB : flasdisk 4 GB : flasdisk 8 GB

= 30 : 45 : 27

= (30 ÷ 3) : (45 ÷ 3) : (27 ÷ 3)

= 10 : 15 : 9


b. Perbandingan ketiga ketiga jenis flashdisk jika telah terjual 5 flashdisk ukuran 2 GB dan 7 flashdisk ukuran 8 GB

= flasdisk 2 GB : flasdisk 4 GB : flasdisk 8 GB

= (30 – 5) : 45 : (27 – 7)

= 25 : 45 : 20

= (25 ÷ 5) : (45 ÷ 5) : (20 ÷ 5)

= 5 : 9 : 4

17. seorang siswa mempunyai flash disk dengan kapasitas penyimpanan sebesar 8 Gb, di dalam flashdisk tersebut diisi beberapa data yaitu 100 foto yang masing-masing foto berukuran 20 MB dan file video sebesar 1.450 MB serta 1 file aplikasi sebesar 4,12 GB. berapakah sisa kapasitas penyimpanan flashdisk siswa tersebut?pke cara ya.​


Jawaban:

3,701,518 MB.

Penjelasan:

Jumlah kapasitas penyimpanan yang digunakan oleh foto adalah 100 foto x 20 MB/foto = 2.000 MB.

Sedangkan jumlah kapasitas penyimpanan yang digunakan oleh video adalah 1.450 MB.

Total kapasitas penyimpanan yang digunakan adalah 2.000 MB + 1.450 MB + 4,12 GB = 7.570 MB + 4,12 GB = 7.570 MB + 4,290,912 MB = 4,298,482 MB.

Sisa kapasitas penyimpanan flashdisk siswa tersebut adalah 8 GB - 4,298,482 MB = 3,701,518 MB.

Jawaban:

Untuk mengetahui sisa kapasitas penyimpanan flash disk siswa tersebut, pertama-tama harus menghitung jumlah ukuran total dari semua file yang ada di dalam flash disk tersebut.

Jumlah ukuran total dari foto-foto tersebut adalah 100 foto x 20 MB/foto = 2.000 MB

Jumlah ukuran total dari semua file yang ada di flash disk tersebut adalah 2.000 MB + 1.450 MB + 4.12 GB = 7.570 MB + 4.12 GB = 4,93 GB.

Kemudian, untuk mengetahui sisa kapasitas penyimpanan flash disk tersebut, bisa mengurangi jumlah ukuran total file dari kapasitas penyimpanan flash disk yaitu 8 GB - 4,93 GB = 3,07 GB. Jadi, sisa kapasitas penyimpanan flash disk siswa tersebut adalah 3,07 GB.

Jika diinginkan, bisa juga mengubah jumlah ukuran total file yang ada di flash disk tersebut ke dalam satuan MB, sehingga sisa kapasitas penyimpanan flash disk siswa tersebut dapat dinyatakan dalam satuan MB juga.

Jumlah ukuran total file yang ada di flash disk tersebut adalah 4,93 GB = 4.930 MB.

Sisa kapasitas penyimpanan flash disk siswa tersebut adalah 8.000 MB - 4.930 MB = 3.070 MB.

28. Diagram Lingkaran Di samping menunjukkan File yang Terdapat di dalam Flashdisk Milik Reta Yang berkapasitas 4 GB (setara dengan 4.000 MB). Flashdisk tersebut diisi dengan File musik,Foto,data buku ajar Matematika,dan data lainya. A.jika Reta ingin Menambahkan File data buku ajar baru yang berkapasitas 750 MB,Apakah kapasitas Flashdisk Milik Reta masih mencukupi?Jelaskan B.Jika Reta Ingin menghapus File Foto,File data buku ajar dan File data lainnya di Flasdisknya,Berapa persen dari keseluruhan File musik yang Harus agar data buku ajar baru dapat Ditambahkan ke dalam Flashdisk?


Diagram lingkaran di samping menunjukkan file yang terdapat di dalam flashdisk milik reta yang berkapasitas 4 GB (setara dengan 4.000 MB). flashdisk tersebut diisi dengan file musik, foto, data buku ajar matematika, dan data lainnya.

A.    Jika Reta ingin menambahkan file data buku ajar baru yang berkapasitas 750 MB, apakah kapasitas flashdisk milik reta masih mencukupi?, jelaskan!. B.    Jika Reta tidak ingin menghapus file foto, file data buku ajar, dan file data lainnya di flashdisknya, berapa persen dari keseluruhan file musik yang harus dihapus agar data buku ajar baru dapat ditambahkan ke dalam flashdisk?

Statistika adalah ilmu (metode ilmiah) yang mempelajari cara-cara mengumpulkan, menyusun, menyajikan, dan menganalisis data serta cara mengambil kesimpulan yang logis sehingga dapat diambil keputusan yang akurat.

Penyajian data dalam bentuk

TabelDiagram batang Diagram garisDiagram lingkaran

Pembahasan

Perhatikan gambar diagram lingkaran yang ada pada lampiran.

A.  Menambahkan file data buku ajar baru berkapasitas 750 MB

Kapasitas kosong = [tex]\frac{10}{100}[/tex] × 4000 MB

                              = 400 MB

Ternyata flasdisk Reta tidak mencukupi apabila diisi dengan file data buku ajar baru berkapasitas 750 MB. Pada kapasitas flasdisk ruang kosong hanya tersedia 10% atau 400 MB, sehingga tidak dapat mencukupi file yg diisikan.

B.  Persen data yang dihapus dari keseluruhan file musik

Kekurangan data (dihapus) = kapasitas file baru - kapsitas ruang kosong

                                             = 750 MB - 400 MB

                                             = 350 MB

Kapasitas file musik = [tex]\frac{20}{100}[/tex] × 4000 MB

                                 = 800 MB

% file data yg dihapus dari keseluruhan file musik

= [tex]\frac{data~ dihapus}{file~ musik}[/tex] × 100 %

= [tex]\frac{350}{800}[/tex] × 100 %

= 43,75 %

Jadi file musik yang harus dihapus supaya mencukupi file baru adalah 43,75 % dari file musik

misal harga flashdisk merah = a , dan harga flashdisk putih = b
6a+4b = 41.000 (pers 1)
sementara b = a + 1.500

untuk b = a+1.500 dimasukkan ke pers 1
6a + 4(a+1.500) = 41.000
6a+4a+6.000 = 41.000
10a + 6.000 = 41.000
10a = 41.000 - 6.000
10a = 35.000
a = 35.000 : 10
a = 3.500

untuk a = 3.500 ==> b = a+1.500 maka b = 3.500+1.500 = 5.000

jadi yang harus dibayar untuk 15a+10b
= 15(3.500) + 10(5.000)
= 52.500 + 50.000
= 102.500
