30. 11c + 2d = 37 5c + 2d = 19


Mapel : Matematika
Bab : Sistem Persamaan Linear Dua Variabel (SPLDV)

11c + 2d = 37
5c + 2d = 19
-------------------  -
        6c = 18
          c = 18/6
          c = 3

   5c + 2d = 19
5(3) + 2d = 19
  15 + 2d = 19
          2d = 19 - 15
          2d = 4
            d = 4/2
            d = 2

HP = { 3, 2 }Pertama variabelnya disamain
11c+5c+2d+2d=37+19 jadi
16c+4d=56
