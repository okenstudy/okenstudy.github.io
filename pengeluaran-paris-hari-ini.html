Pembahasan

Bilangan Bulat

[tex]\:[/tex]

Bilangan bulat adalah bilangan utuh (bukan pecahan) yang terdiri dari atas bilangan nol, positif, dan negatif. Bilangan nol menyatakan banyaknya anggota himpunan yng tidak mempunyai anggota sama sekali, atau himpunan kosong. Banyaknya anggota himpunan kosong adalah nol.

[tex]\:[/tex]

Termasuk Bilangan Bulat

[tex]\:[/tex]

Bilangan PositifBilangan Negatif

[tex]\:[/tex]

Bilangan Positif

[tex]\:[/tex]

Bilangan positif merupakan bilangan yang sama saja dengan bilangan asli. Bilangan positif merupakan lawan dari bilangan negatif karena bilangan positif semakin besar angkanya maka semakin besar juga jumlahnya. Bilangan Positif / Asli tersebut adalah

[tex]\:[/tex]

{ 1, 2, 3, 4, 5, .... Dan Seterusnya }

[tex]\:[/tex]

Bilangan Negatif

[tex]\:[/tex]

Bilangan Negatif merupakan bilangan yang diawali dengan tanda ( - ) pada depan angka tanda tersebut dibaca dengan minus, Bilangan negatif ini semakin besar jumlah angkanya maka semakin kecil hitungannya itulah mengapa bilangan megatif dan positif saling berlawanan. Bilangan negatif terdiri dari { -1, -2, -3 -4, -5, Dan seterusnya }

[tex]\:[/tex]

Rumus Pertambahan

[tex]\:[/tex]

➽ Positif + Positif = Positif ➽ Negatif + Negatif = Negatif➽ Positif + Negatif = Negatif➽ Negatif + Positif = Positif

[tex]\:[/tex]

Rumus Pengurangan

[tex]\:[/tex]

➽ Positif - Positif = Negatif➽ Negatif - Negatif = Positif➽ Positif - Negatif = Positif➽ Negatif - Positif = Negatif

[tex]\:[/tex]

Rumus Perkalian

[tex]\:[/tex]

➽ Positif × Positif = Positif➽ Negatif × Negatif = Positif➽ Positif × Negatif = Negatif➽ Negatif × Positif = Negatif

[tex]\:[/tex]

Rumus Pembagian

[tex]\:[/tex]

➽ Positif ÷ Positif = Positif➽ Negatif ÷ Negatif = Positif➽ Positif ÷ Negatif = Negatif➽ Negatif ÷ Positif = Negatif

Bilangan bulat adalah himpunan bilangan yang mencakup bilangan cacah , bilangan asli , bilangan nol , bilangan satu, bilangan prima, bilangan komposit dan bilangan negatif. Atau kesimpulan lain dari bilangan bulat adalah himpunan bilangan yang mencakup seluruh bilangan, kecuali bilangan imajiner , irrasional dan pecahan.

[tex]\:[/tex]

Bilangan Cacah

[tex]\:[/tex]

Bilngan Cacah adalah bilangan yang tidak termasuk angka negatif, Bilangan cacah terdiri dari angka awalan nol / ( 0 ). Hampir sama dengan bilangan Asli namun bilangan asli tidak terdiri dari nol. Bilangan Cacah terdiri dari { 0, 1, 2, 3, 4, .... Dan Seterusnya }

[tex]\:[/tex]

Bilangan Asli

[tex]\:[/tex]

Bilangan Asli adalah bilangan yang sama saja dengan bilangan positif yaitu dimulai dari angka 1 kemudian angka seterusnya. Bilangan Asli terdiri dari { 1, 2, 3, 4, 5, .... Dan seterusnya }

[tex]\:[/tex]

Bilangan Prima

[tex]\:[/tex]

Bilangan Prima adalah bilangan asli yang lebih besar dari satu dan mempunyai tepat dua faktor, yaitu bilangan satu dan bilangan itu sendiri. Bilangan prima terdiri dari { 2, 3, 5, 7, 11 .... Dan seterusnya }
