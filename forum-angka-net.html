Kelas = BNet id = 142.198Host id = 243.202Subnet mask = 255.255.0.0Angka binary = 10001110.11000110.11000110.11001010

Untuk melakukan konversi bilangan (dari desimal ke biner), dapat dibantu program pengolah angka seperti Ms. Excel 2010. Amatilah lampiran untuk mengetahui rumus konversinya (gambar atas).
