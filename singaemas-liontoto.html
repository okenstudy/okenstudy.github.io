Jumlah ayam

[tex] = \frac{9}{9 + 5} \times 70 \\ = \frac{9}{14} \times 70 \\ = 9 \times 5 \\ = 45[/tex]

Jumlah itik

[tex] = \frac{5}{9 + 5} \times 70 \\ = \frac{5}{14} \times 70 \\ = 5 \times 5 \\ = 25[/tex]

Jadi selisihnya adalah 45 - 25 = 20

7. Sebuah saham PT. Sinarmas disepakati banyak analis tidak akan membagikan dividen. Jika harganya tahun depan adalah Rp 4000 sedangkan harga pasarnya sekarang adalah Rp 3200, berapakah return yang diharapkan investor?


Untuk menghitung return yang diharapkan investor, kita dapat menggunakan rumus return yang diharapkan (expected return) yaitu:

expected return = (harga tahun depan - harga saat ini) / harga saat ini

Dengan menggunakan rumus tersebut, maka return yang diharapkan investor adalah:

expected return = (4000 - 3200) / 3200

expected return = 800 / 3200

expected return = 0,25 atau 25%

Jadi, return yang diharapkan investor adalah 25%.
