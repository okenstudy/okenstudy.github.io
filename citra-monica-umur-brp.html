29. umur amir : umur bandi = 5:3. sedangkan umur bandi : citra = 4:5 jika selisih umur amir dan citra 10 tahun, maka jumlah umur mereka semua adalah...


Misal :
Umur amir = A
Umur bandi = B
Umur citra = C

A : B = 5 : 3, maka A = 5/3 B
B : C = 4 : 5, maka B = 4/5 C
A - C = 10, maka C = A - 10

*Menentukan nilai B
B = 4/5 C
= 4/5 (A-10)
= (4A - 40)/5
*Menentukan nilai A
A = 5/3 B
A = 5/3 [(4A-40)/5]
3A = 4A-40
A=40

Substitusikan nilai A untuk mengetahui nilai B
B= (4A - 40)/5
= 24

Nah C = A - 10, maka C = 30

Jadi, total umur A+B+C = 40+24+30=94
