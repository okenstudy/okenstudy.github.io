Bebegig Sukamantri nyaeta salahsahiji rupa kasenian has anu aya di daerah Ciamis, anu perenahna nyaeta di Desa Cempaka, Kacamatan Sukamantri, Kabupaten Ciamis. Kasenian Bebegig Sukamantri biasa dipintonkeun saperti dina acar agustusan, hiburan dina acara sunatan atawa dina acar ulang taun hiji daerah.

Sedengkeun ari bebegig sawah mah mangrupa jajalmaan anu rupana pikasieuneun anu ditancebkeun di tengah sawah atawa di balong anu tujuannana pikeun nyingsieunan manuk. Bebegig Sukamantri mangrupa jalma asli anu make kedok gede anu rupana pikasieuneun jeung buukna cawigwig.

1. Bebegig Sukamantri nyaeta salahsahiji rupa kasenian has anu aya di daerah Ciamis. Kasenian Bebegig Sukamantri biasa dipintonkeun saperti dina acara agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah. 2. Bebegig Sukamantri mangrupa jalma asli anu make kedok gede anu rupana pikasieuneun jeung buukna cawigwig.

3. Ngamaenkeun Bebegig Sukamantri mah kudu gede tanaga, sabab topengna wae beuratna kurang leuwih 20 kg, eta teh beurat topengna wungkul, can kaitung beurat pakean anu sejenna anu beurat oge.

4. Mimiti medalna kasenian Bebegig Sukamantri teh nyaeta ti Kabupaten Ciamis, anu perenahna nyaeta ti Desa Cempaka, Kacamatan Sukamantri, Kabupaten Ciamis.

5. Topeng Bebegig Sukamantri teh mangrupa gambaran kagagahan Prabu Sampulur.

6. Kasenian Bebegig Sukamantri biasa dipintonkeun saperti dina acara agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah.

7. Bebegig Sukamantri teh mangrupakeun bagian tina ritual dina upacara pikeun ngusir roh anu jahat. Kusabab ayeuna warga Ciamis geus ngagem agama Islam, kasenian Bebegig dirobah tina upacara mistik ngusir roh jahat jadi kasenian pikeun ngahibur masarakat Ciamis saperti dina acara agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah.
