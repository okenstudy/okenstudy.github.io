Baris ke-1 warna merah

Baris ke-2 warna hijau

Baris ke-3 warna putih

Baris ke-4 warna merah muda

Baris ke-5 warna biru

Baris ke-6 warna kuning

Karena pola berulang, maka pada baris ke-7, motifnya kembali ke warna merah, baris ke-8 warna hijau, dst. sampai baris ke-39.

Misal, Un menyatakan warna yang muncul pada baris ke-n, maka

U1 = merah

U2 = hijau

U3 = putih

U4 = merah muda

U5 = biru

U6 = kuning

Ditanyakan: Un = putih, n = ?

Terbentuk pola barisan bilangan aritmatika pada masing-masing warna dengan selisih/beda 6.

Diperoleh data sebagai berikut;

a) Motif warna merah muncul pada baris ke-1, 7, 13, 19, 25, 31, 37, maka

U1 = U7 = U13 = U19 = U25 = U31 = U37 = merah

b) Motif warna hijau muncul pada baris ke-2, 8, 14, 20, 26, 32, 38, maka

U2 = U8 = U14 = U20 = U26 = U32 = U38 = hijau

c) Motif warna putih muncul pada baris ke-3, 9, 15, 21, 27, 33, 39, maka

U3 = U9 = U15 = U21 = U27 = U33 = U39 = putih

d) Motif warna merah muda muncul pada baris ke-4, 10, 16, 22, 28, 34, maka

U4 = U10 = U16 = U22 = U28 = U34 = merah muda

e) Motif warna biru muncul pada baris ke-5, 11, 17, 23, 29, 35, maka

U5 = U11 = U17 = U23 = U29 = U35 = biru

f) Motif warna kuning muncul pada baris ke-6, 12, 18, 24, 30, 36, maka

U6 = U12 = U18 = U24 = U30 = U36 = kuning

Berdasarkan data di atas, dengan pola tersebut,

maka motif berwarna putih akan muncul pada baris ke-3, 9, 15, 21, 27, 33, 39.

Mungkinkah motif berwarna putih muncul pada baris ke-25 dan ke-37?

Dengan pola warna tersebut, tidak mungkin warna putih muncul di baris ke-25 dan ke-37, karena pada baris ke-25 dan ke-37 akan muncul motif warna merah. Warna putih bisa muncul di baris ke-25 dan ke-37 jika menjadi warna pada baris pertama.
