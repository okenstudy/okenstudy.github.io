Dengan menggunakan command prompt kita bisa melakukan perintah copy paste dan perintah-perintah lainnya dengan mengetikan perintah tersebut pada tampilan command prompt.

Adapun beberapa cara untuk mengakses command prompt adalah sebagai berikut

1. Menggunakan start menu, kita bisa menjalankan command prompt melalui start menu kemudian kita ketik pada kolom dengan kata command prompt.

2. Cara untuk mengakses yang kedua adalah menggunakan run, tekan tombol windows dan R secara bersamaan, maka akan muncul kolom run, kemudian ketik cmd dan enter.

3. Cara yang terakhir adalah memanfaatkan file .bat file ini bisa kita buat melalui notepad, buka notepad kita kemudian ketik cmd, save pada lokasi yang kita inginkan , beri nama sesuai keinginan namun pada pilihan tipe file kita pilih all file, kemudian beri ektensi .bat. Klik dua kali file tersebut maka command prompt akan terbuka

10. apa bedanya command line sama command prompt??


Command line merupakan tempat penulisan
command yang terdapat pada Linux.
Perbedaannya dapat dilihat di bawah :
1. Direktorinya dalam bentuk file
2. Prompt ada 2 yaitu dapat berfungsi sebagai
user ($) atau juga sebagai root (#).
3. Banyak sesi yang digunakan untuk
menyelesaikan satu perintah.
4. Mode yang digunakan ada dua,yaitu mode
grafis dengan mode non grafis (tty)
Command Prompt Windows
Command Prompt merupakan tempat penulisan
command yang terdapat pada Windows.
Perbedaannya juga dapat dilihat di bawah :
1. Direktorinya dalam bentuk drive
2. Prompt nya hanya drive
3. Hanya satu sesi yang digunakan untuk
menyelesaikan satu perintah
4. Mode yang digunakan mode grafis ( desktop)
