6. nilai varians dari data 122, 125, 120, 123, 124, 124​


Nilai varians dari data 122, 125, 120, 123, 124, 124​ adalah 2,67

Pendahuluan

Rata-rata adalah jumlah semua nilai data dibagi dengan banyak data.

[tex]\boxed {\overline {x} = \frac{x_1~+~ x_2~ +~ x_3~ +~ ...~ +~ x_n}{n}}[/tex]

Varian (ragam) data

[tex]\boxed {S^{2} = \frac{1}{n} \Sigma^n_{i = 1} (x_i - \bar{x})^2}[/tex]

Pembahasan

Diketahui :

Data : 122, 125, 120, 123, 124, 124​

Banyak data (n) = 6

Ditanya :

Nilai varian ?

Penyelesaian :

Menentukan rata-rata

[tex]\displaystyle \overline {x} = \frac{x_1~+~ x_2~ +~ x_3~ +~ ...~ +~ x_n}{n}[/tex]

[tex]\displaystyle \overline {x} = \frac{122 + 125 + 120 + 123 + 124 + 124}{6}[/tex]

[tex]\displaystyle \overline {x} = \frac{738}{6}[/tex]

x = 123

Menentukan ragam (Varian) dari data

[tex]\boxed {S^{2} = \frac{1}{n} \Sigma^n_{i = 1} (x_i - \bar{x})^2}[/tex]

S² = ¹/₆ [ (122 - 123)² + (125 - 123)² + (120 - 123)² + (123 - 123)² + (124 - 123)² + (124 - 123)² ]

S² = ¹/₆ [ (-1)² + 2² + (-3)² + 0² + 1² + 1² ]

S² = ¹/₆ [1 + 4 + 9 + 0 + 1 + 1]

S² = ¹/₆ × 16

S² = 2,67

Jadi nilai varians dari data 122, 125, 120, 123, 124, 124​ adalah 2,67.
