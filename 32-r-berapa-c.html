30. 32° R =...° C=...°F​


Pertanyaan : 32° R =...° C=...°F

Jawaban :

====| Rumus |====

• C = Celcius

• R = Reamur

• F = Fahrenheit

• K = Kelvin

[tex] \frac{c}{5} = \frac{r}{4} = \frac{f - 32}{9} = \frac{k - 273}{5} [/tex]

32° R = ... C

[tex] = > \frac{c}{5} = \frac{32}{4} \\ = > 4c = 160 \\ = > c = 40[/tex]

32° R = ... F

[tex] = > \frac{f - 32}{5} = \frac{32}{4} \\ = > \frac{f - 32}{5} = 8 \\ = > f - 32 = 40 \\ = > f = 40 - 32 \\ = > f = 8[/tex]

Jadi 32° R = 40° C = 8° F
